# 1. 快速入门
- 如果文件不存在，它会在内存里先创建一个临时的，等你保存（输入 `:w`）时才真正写入硬盘。
- 如果你是 Linux 新手，用 `vim` 创建文件后可能会发现“退不出来”。记住这套**“逃生口诀”**：
	1. 按 **`Esc`** 键（确保退出编辑模式）。
	2. 输入 **`:wq`**（保存并退出）或者更极速的 **`ZZ`**。
	3. 或者输入 **`:q!`**（不保存，强制退出）或者按 **`ZQ`** 挥一挥衣袖不带走一片云彩。
- 如果你只是想快速看一眼文件里写了什么，而不想进去编辑，可以试试 `cat xxx.txt`
- **`vim vimtutor`**：展示官方基础教程。

# 2. Vim 实战教程（把键盘按出火星子）

## 2.1 盲打指法：上下左右移动
- **`h`**：光标向左（指尖向左）
- **`j`**：光标向下（食指向下，最常用的按键）
- **`k`**：光标向上（中指向上）
- **`l`**：光标向右（无名指向右）
*(注意：CapsLock 键绝对不能按下，因为大写字母会触发完全不同的命令！)*

## 2.2 基础模式切换（怎么开始写代码）
在普通模式下，你必须“进入”插入模式才能像在普通记事本里一样打字：
- **`i`** / **`I`**：`i` 是在光标当前位置直接开写；而 `I` 则是“我就要第一名”，直接跳到这行最前面开始写。
- **`a`** / **`A`**：`a` 是在光标后面追加；`A` 是冲到行尾，方便你在一句话结尾加分号 `;`。
- **`o`** / **`O`**：`o` 是在光标下方优雅地新开一行并让你写；`O` 则是在光标上方强行插入一行。
- **换字符换大小写**：用 **`r`** 可以单换当前的一个字；碰到一段字母大变小/小变大？选中后按 **`~`** 变身。
- **合并多行**：按 **`J`** 把下一行“吸”上来跟现在这行贴一起。
- **撤销和重做口诀**：写错了怎么办？按 **`u`** 撤销；如果撤销成了白板，按 **`Ctrl + R`** 把未来找回来。

## 2.3 快速移动与跳转（在代码中穿梭）
如果你觉得光按 `hljk` 太慢了，试试这些飞天遁地的招数：
- **单词跳跃**：**`w`** / **`W`** 往后跳一个词（大写忽视标点）；**`b`** / **`B`** 往前倒退一个词；**`e`** / **`E`** 飞跃到单词末尾。
- **行内突刺**：**`0`** 彻底回到最左边；**`^`** 回到第一个写字的字符前（软行首）；**`$`** 瞬间跳到行尾。你想精确去这一行第 `n` 列？数字加 **`|`**。
- **首尾极速转移**：**`gg`** 一键飞回文件第一行；**`G`** 一键飞跃到文件最后一行。如果老板让你改第 35 行的 Bug，就敲 **`35G`**。
- **找括号兄弟**：停在半个 `{` 上，按 **`%`**，它会自动帮你跳到那个对应的 `}` 上！
- **行内心眼查找**：按 **`f` + 字母** 往前找这个字母并飞过去（比如 `f;` 找分号）；大写的 **`F`** 是往回找；小写 **`t`** 是飞过去但是停在目标的前一个位置（类似刹车）。按 **`;`** 和 **`,`** 可以顺/逆方向重复刚刚的查找。

## 2.4 删除、修改与替换（删库跑路好手）
千万不要在普通模式下疯狂按 Backspace 或 Delete！
- **删字符**：**`x`** 删掉光标所在的字（相当于 Delete），**`X`** 删光标前面的字（等于 Backspace）。
- **删词**：**`dw`** 从这里往后死死删除到下一个词的开头；**`de`** 删到当前词末尾；**`db`** 往回删词头；**`diw`** （最常用）无情噶掉这整个单词，但留着两边空格；**`daw`** （更狠）连词带空格一起噶掉。
- **删行**：**`dd`** 一刀切掉光标这一整行；**`d$`** 删到行尾；**`d^`** 删到首非空字符。
- **修改到尾巴**：**`C`** 把这行后面的全删了并且立刻开始打字；**`cc`** 或者 **`S`** 整行删空立刻打字。如果你想改当前词的结尾？**`ce`**。或者针对一个字符修改？**`s`** 删它并进入打字。如果你想修改 `()` 大括号里的东西？**`ci{`** 一键清空内部开写！
- **删到什么前面**：**`dt` + 字符** 删到目标字符前；修改则是 **`ct` + 字符**。
- **核弹级替换大招**：`:[范围]s/旧/新/[标志]` 比如在全局查旧换新 `:%s/Old/New/g` (带 `c` 可以一个个确认！加上 `\<旧\>` 可严格匹配整个单词)

## 2.5 复制与粘贴（CV工程师的荣誉）
Vim 的复制叫 `yank` (拽下来)，粘贴叫 `put` (放上去)：
- **复制整行的快乐**：**`yy`** 或者 **`Y`** 直接把这一行复制走。想复制 3 行？就打 **`3yy`**。
- **复制单词与片段**：**`yw`** 复制一个词；**`y$`** 一直复制到行尾；**`y^`** 复制到行首。特别提一下 **`0y$`**，从头复制到尾但不复制多余的换行符，非常好用！
- **粘贴口诀**：**`p`** 贴在光标后面或下一行，**`P`** 贴在前面或上一行。
- **缩进调整**：全选后 **`>>`** 往右缩进，**`<<`** 往左缩退。
- **一锅端**：**`:%y`** 复制全文；**`:%d`** 删除全文；**`ggvG`** (或者 `Gvgg`) 全选文章高亮。
- **与系统外交互**：如果你想发微信或者其他软件？按 **`:%y+`** 复制全文到剪贴板，或者用 **`"+p`** 从外界粘贴进来（因为你装了外挂，这步被后面的大招取代了）。

## 2.6 查找、滚动与定位（火眼金睛与乾坤大挪移）
要在几万行代码里找东西、看东西：
- **按词搜**：按 **`/`** 输入你想找的字，回车定位。按 **`n`** 找下一个，**`N`** 找上一个。要是向上搜索按 **`?`**。更快的办法：光标指在一个单词上，按 **`*`** 自动向下高亮查找，按 **`#`** 自动向上查找！
- **屏幕视点掌控（三连拔）**：如果你想不用滚动去找：**`H`** 飞到屏幕顶部，**`M`** 飞跃到屏幕中间，**`L`** 直接到屏幕底部。
- **对焦神技**：觉得光标在屏幕下面看得很难受？**`zz`** 把它直接拽到屏幕中央；**`zt`** 怼到顶；**`zb`** 怼到底。
- **大力翻书**：翻页找 **`Ctrl + F`** (下翻一页)、**`Ctrl + B`** (上翻一页)；或者是温和的 **`Ctrl + D / U`** 半页翻；也可以只滚一行 **`Ctrl + E / Y`**。按 **`Ctrl + G`** 可以炫耀一下查当前看了多少字的文件信息。

## 2.7 文本对象与结构化操作（精准狙击战）
代码总是成双成对出现的呀（比如双引号，括号）：
- 想复制别人括号/引号里的心血？**`yi(`** (复制圆括号内的代码)，**`ya(`** (连圆括号一起全拿走)，**`yi"`** (复制双引号内的)。同理还有复制 HTML 标签里的心血 **`yit`**。
- **选词技巧**：**`yiw`** 复制没标点的小单词，大写的 **`yiW`** 连标点一块儿拿走大单词。
- 用 **`v%`** 可以帅气地高亮选中当前从老巢 `{` 一路到底 `}`；删除用 **`d%`**，复制用 **`y%`**。

## 2.8 格式化与选区（代码强迫症福音）
- **一键排版缩进**：按下 **`=`**；如果是排版当前行就 **`==`**，要从当前一直到结束直接打 **`=G`**。全文乱套直接 **`gg=G`**。
- 如果就大括号里的东西不对劲，你就在括号里按 **`=i{`**。
- **二次高亮找回**：刚刚选中文本按了其他键高亮没了？按 **`gv`** 一下子原样帮你框出来！
- 怎么高亮选下一个刚才搜过的东西？用 **`gn`**。

## 2.9 寄存器、标记与记忆术（时空跳跃）
每次删东西其实它是跑到寄存器里存下来了：
- **查账**：敲 **`:reg`** 可以看到你剪贴了什么。刚才那次复制放进了 **`0`**，最近的 9 次删除依次回荡在 **`1-9`**。当前文档名叫 **`%`**。
- **种眼与传送**：逛到重要战线按 **`ma`** 放个名叫 `a` 的神眼标记。然后你去游山玩水去了，按下 **`'a`** 瞬间传送到眼所在行首！按 **`` `a ``** 则是精确到眼摆放的那个字！也可以敲 **`` `` ``** 跳回上一次的位置，或者 **`` `. ``** 回到你刚才敲键盘的地方。用 **`:marks`** 查所有的眼。
- **穿梭时空**：按 **`Ctrl + O`** 可以退回到上一页代码，再按 **`Ctrl + I`** 像浏览器前进键一样再穿回来。

## 2.10 打通任督二脉：多终端管理
当你在几个文件中横跳：
- **看后台文档**：敲 **`:ls`** 显示你开了哪些大招文件，**`:b 编号`** 直接翻牌子，**`:bn` / `:bp`** 是下一章或上一章轮换。
- **开挂看多个（标签页）**：**`gt` / `gT`** 就是 Chrome 浏览器的前进后退标签页，还可以 **`3 gt`** 直奔第3个。
- **真正的黑客分屏**：水平切开屏幕 **`Ctrl + W + s`**；垂直切 **`Ctrl + W + v`**。你想在分屏里跳跃？按 **`Ctrl + W + h/j/k/l`** 换人。不玩了就 **`Ctrl + W + q`** 关掉或 **`Ctrl + W + o`** 让他走人只有你留下。
- **调整身材**：利用 **`Ctrl + W + >/<`** 调节宽窄，按 **`Ctrl + W + =`** 直接让他们均匀等分！或者 **`Ctrl + W + _|`** 霸道充满当前屏。可以用大写 **`Ctrl + W + H/J/K/L`** 或者轮换的 **`r`** 把窗口彻底换边。

## 2.11 奇怪的知识增加了（专用的微调）
- **插入模式秘技**：打错字还在用 Backspace？按 **`Ctrl + W`** 可以删前一个单词！**`Ctrl + U`** 删光标到行首的内容。要填单词？按 **`Ctrl + N / P`** 会给你代码补全弹窗提示。敲 **`Ctrl + T / D`** 能快速控制当前行缩进。
- 敲着字发现你想删掉上面两行的东西怎么办？千万别退出来！按 **`Ctrl + O`** 临时切回无敌状态释放一招魔法，敲完你会奇迹般直接接续当前的打字状态。
- **命令行模式秘技**：打命令发现很长，可以按 **`Ctrl + F`** 把历史命令像记事本一样展开编辑修改。你想把刚复制的东西粘贴进命令里？可以在命令里打 **`Ctrl + R + 寄存器名(如")`**，如果想把光标下指着的那个单词搜寻直接拽进来在命令行，用 **`Ctrl + R + Ctrl + W`**。

# 3. 终极大招（私人 VSCode Vim 定制挂）
你已经在 VSCode 的 `settings.json` 把自己改造成了战斗机器。这是一套只属于你的最强连招：

### ⚙️ 外挂被动技能
- **全局剪贴板同频**：你设置了 `vim.useSystemClipboard: true`。从此 Vim 里复制的任何东西全部飞进电脑系统的 Ctrl+C，外面粘进来的 Ctrl+V 也能随手用 `p` 喷出来。
- **硬核空格排版**：你的缩进完全被调教成了严格插入空格取代 Tab (`editor.insertSpaces: true` 且禁掉了自带 `detectIndentation: false`)。

### 🤺 普通模式下：天下武功唯快不破（Space 触发）
- **御风飞行**：**`<Space> j`** 与 **`<Space> k`** 直接一口气往上往下暴飞 `20` 行！
- **极速搬运**：虽然有被动，但还是加了 **`<Space> y`** 与 **`<Space> p`**，确保任何情况都是走 `"+` 挂着系统剪贴板拿来拿去。
- **刺客隐身**：搜完字背景高亮瞎眼？按下 **`<Space> n l`** (`:nohl`)，一键熄灭高亮，深藏功与名。
- **三指救命**：平时还在敲冒号？**`<Space> w`** 直接调起底层引擎保存。**`<Space> q`** 退出，**`<Space> x`** 存退合一，连 Shift 都不用按了。
- **引力连击**：按 **`<Space> '`** 将下一行吸到你这句话的屁股后面贴着（自动杀掉空格退回常态）。
- **核弹抢夺全选（强推）**：普通全选嫌累？**`<Space> a c`** 一手抓爆全文。你想把这页代码丢给别人？**`<Space> a y`**，先立原位眼，一键全选复制系统，接着瞬传回立眼的位置——就好像你啥都没干过，但东西已经在了！带空格的连斩版本 **`<Space> a <Space> y`** 更是明目张胆拷到系统里。

### 📝 插入与可视模式（花式编辑法）
- **神仙归位**：你在专心打字想休息一下？无需伸手去构那个极偏远的 Esc 键，直接左手连点双扣 **`jj`** 立马优雅撤出来。
- **HTML 补全仙术**：如果你在写前端文本，插入模式敲下 **`,,s`** 瞬间长出 `<sub></sub>` 下标标签还贴心把你光标框在中间。如果按 **`,,p`** 则瞬间长出 `<p></p>` 同样塞你进里面直接开讲。
- **框选即包裹（魔法包衣）**：选中一堆散字后，按 **`,,s`** 直接强行将他们套在 `<sub>` 里面。如果你选中文本后敲下私人招式 **`<Space> 8`**，这堆字立刻两边加星号变加粗！敲下 **`<Space> 9`** 被 `()` 锁上，**`<Space> [`** 就用防弹车盖 `[]` 强行包裹！
