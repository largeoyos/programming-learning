### 🔑 这就引出了本节最核心的概念：**参考方向**

我来给你一个精确的定义，你体会一下：

> **参考方向（Reference Direction）**：在分析电路前，人为假设的电压或电流的正方向。它 **不一定** 是实际方向，但一旦设定就不能改。计算结果的正负号告诉我们实际方向和参考方向的关系：
> 
> - 结果为正 → 实际方向 = 参考方向
> - 结果为负 → 实际方向 ↔ 参考方向（相反）

### 📝 参考方向的两种表示法

**电流的参考方向**：用箭头标在导线上

```
     I
  ───→───
```

这个箭头的意思是："**我假设**电流从左流到右"。

**电压的参考方向**：用 "+" "−" 号标在元件两端，或者用双下标

```
    +  U  −
  ───[R]───
```

这个的意思是："**我假设**左端电位高，右端电位低"。

---

### 🌟 参考方向最妙的用法：关联参考方向

当一个元件的 **电压参考方向** 和 **电流参考方向** 满足 "**电流从电压的"+" 端流入**" 时，我们称它们为 **关联参考方向**。

```
     I
  ───→───
    +  U  −
  ───[R]───
```

**为什么要搞这个概念？** 因为它能让我们的公式变得超级统一：

在 **关联参考方向** 下：

$P = UI$

- P > 0：元件**吸收**功率（比如电阻发热）
- P < 0：元件**释放**功率（比如电池放电）

在 **非关联参考方向** 下：

$P = -UI$

> 💡 这样一来，不管是电阻、电池、还是未知的黑箱，我们都用同一个公式算功率，**靠正负号自动区分吸收/释放**。

## 1.2 基尔霍夫定律（KCL 和 KVL）

这是你高中没学过的内容，也是全课程的**两大基石**。后面所有的电路分析方法（支路法、回路法、节点法）本质上都是在反复使用这两条定律。

### 先建立几个基础概念

看一个电路的"骨架"，需要先认识三个词：

```
         a ────[R1]──── b
         │              │
        [R2]          [R3]
         │              │
         c ────[R4]──── d
```

- **支路（Branch）**：每个二端元件就是一条支路。上图有 4 条支路（R1、R2、R3、R4）
- **节点（Node）**：支路的连接点。上图有 4 个节点（a、b、c、d）
- **回路（Loop）**：从某节点出发，沿支路走一圈回到起点，中间不重复经过节点。上图的 a→b→d→c→a 就是一个回路

> ⚠️ **注意**：在电路课里，"节点"有时指**广义节点**——几条导线直接相连的整个连通块算**一个**节点。比如下面这个：
> 
> ```
>   ──┬──┬──   
>     │  │       这三条线连在一起，只算 1 个节点
>   ──┘  │
>        │
> ```

---

### KCL：基尔霍夫电流定律

#### 物理本质

电荷守恒。电荷不会在节点里凭空产生，也不会凭空消失。

**换句话说**：流进一个节点的电流总量 = 流出该节点的电流总量

#### 数学表达

对任意节点，取"流入为正，流出为负"（或反过来，但必须统一），则：

$\sum_{k} i_k = 0$

#### 看个具体例子

```
          I1 ↓
           │
    I2 →───┼───→ I3
           │
           ↓ I4
```

节点处 KCL：**流入 = 流出**

$I_1 + I_2 = I_3 + I_4$

或者统一写成"流入为正"的形式：

$I_1 + I_2 - I_3 - I_4 = 0$

#### KCL 的推广（重要！）

KCL 不仅对单个节点成立，**对任意一个"闭合面"（高斯面）都成立**。

```
        ┌────────────────┐
        │    I1 →  ┌─┐   │  ← I3
        │          │ │   │
        │    I2 →  └─┘   │  ← I4
        └────────────────┘
              虚线闭合面
```

流入这个框的电流 = 流出的电流：$I_1 + I_2 = I_3 + I_4$

这个推广在分析复杂电路时非常好用——**你可以把一个复杂的子电路当成一个大"节点"来处理**。

---

### KVL：基尔霍夫电压定律

#### 物理本质

能量守恒。沿任意闭合回路绕一圈，电位的升降必然抵消（否则就意味着电子绕一圈能白赚能量，永动机就成立了）。

#### 数学表达

沿任意闭合回路绕一圈，各段电压的代数和为零：

$\sum_{k} u_k = 0$

#### 怎么定代数和的"正负"？

这是 KVL 的操作难点，我给你一个**万能规则**：

> **沿绕行方向前进时，从"−"跨到"+"取正号，从"+"跨到"−"取负号**

（也可以反过来约定，只要全程统一）

#### 看个例子

```
         ┌──→ 绕行方向（顺时针）
         │
      ┌──┴──────────────┐
      │ +           +   │
    U1│             │U2 │
      │ −           −   │
      └────[R]──────────┘
         +    UR   −
```

假设顺时针绕一圈，从左下角出发：

- 跨过 U1：从"−"到"+" → **+U1**
- 跨过 U2：从"+"到"−" → **−U2**
- 跨过 R：从"+"到"−" → **−UR**

KVL 方程：

$U_1 - U_2 - U_R = 0$

即 $U_R = U_1 - U_2$

---

### 🎯 一个完整的例题（把 KCL+KVL 用起来）

```
        I1      a       I2
    ┌───→────┬───→─────┐
    │        │         │
   (+)       │        (+)
   10V      [R2]      4V
    │       2Ω         │
   (−)       │        (−)
    │        │         │
    └────────┴─────────┘
           b (地)
         
         R1=1Ω     R3=1Ω
    （R1在10V和a之间，R3在a和4V之间）
```

重新用更清楚的图描述：左边 10V 电源通过 R1=1Ω 接到节点 a；节点 a 通过 R2=2Ω 接到地 b；右边 4V 电源通过 R3=1Ω 也接到节点 a；电源另一端都接到地。

**设电流参考方向**：

- I1：从 10V 经 R1 流向 a（向右）
- I2：从 4V 经 R3 流向 a（向左）
- I3：从 a 经 R2 流向地（向下）

**节点 a 的 KCL**：流入 = 流出

$I_1 + I_2 = I_3 \quad \text{...①}$

**左回路 KVL**（10V、R1、R2 组成的回路，顺时针绕）：

$10 - I_1 \cdot 1 - I_3 \cdot 2 = 0 \quad \text{...②}$

**右回路 KVL**（4V、R3、R2 组成的回路，逆时针绕）：

$4 - I_2 \cdot 1 - I_3 \cdot 2 = 0 \quad \text{...③}$

三个方程三个未知数，解出：

- $I_1 = 3.2\text{A}$
- $I_2 = -0.4\text{A}$（负号说明实际方向与假设相反，电流其实从 a 流向 4V 电源——4V 被"倒灌"充电了！）
- $I_3 = 2.8\text{A}$

### 🔑 本节你需要记住的几件事

1. **KCL = 电荷守恒**（节点处 $\sum i = 0$，可推广到任意闭合面）
2. **KVL = 能量守恒**（回路中 $\sum u = 0$）
3. **两个定律都和元件是什么无关**——电阻、电容、电感、电源、二极管都适用
4. **正负号的关键是参考方向**：先设方向，再列方程，负号自己会说话

### 一、电阻元件

#### 定义

电阻是一种**耗能元件**，它的伏安特性（VCR，Voltage-Current Relation）由欧姆定律描述。

**在关联参考方向下**：

$u = Ri \quad \text{或} \quad i = Gu$

其中 G = 1/R 是**电导**（单位：西门子 S）。

#### 高中没强调的几个点

**① 非关联参考方向下要加负号**：

```
     I →
   ────────
   −  U  +
   ──[R]──
```

此时：$u = -Ri$

**② 电阻的功率恒为非负**：

$P = ui = Ri^2 = Gu^2 \geq 0$

电阻永远在消耗功率，不会释放。这是它作为**无源元件**的本质。

**③ 线性 vs 非线性**：

- **线性电阻**：R 是常数（本课程默认都是这种）
- **非线性电阻**：R 随 u 或 i 变化（如二极管）

**④ 两个极端**：

- R → 0：**短路**（两端电压恒为 0，电流任意）
- R → ∞：**开路**（电流恒为 0，电压任意）

---

### 二、独立电源

#### 理想电压源

**定义**：两端电压**强制**为某个定值（或定函数）$u_s(t)$，与流过的电流**无关**。

```
      +
    ┌─┴─┐
    │u_s│         符号：带 + - 的圆圈
    └─┬─┘
      −
```

**关键性质**：

- 电流由**外电路**决定，可以任意大小、任意方向
- **不能短路**！短路会导致 $u_s = 0$，与定义矛盾（物理上意味着无穷大电流）

#### 理想电流源

**定义**：流出的电流**强制**为某个定值（或定函数）$i_s(t)$，与两端电压**无关**。

```
     ↑
    ┌┴┐
    │ │         符号：带箭头的圆圈
    │i│         箭头方向 = 电流流出方向
    │s│
    └┬┘
```

**关键性质**：

- 电压由**外电路**决定
- **不能开路**！开路意味着 $i_s = 0$，矛盾（物理上意味着无穷大电压）

#### 一个重要的思维点

独立源的**"倔强"**是它的本质：

- 电压源"咬定"电压不放，电流任你外电路摆布
- 电流源"咬定"电流不放，电压任你外电路摆布

> 💡 后面做戴维南/诺顿等效、电路化简时，这两种"倔强"经常需要你灵活转化。

---

### 三、受控源（重点！）

这是你列出的难点之一，我们慢慢讲。

#### 为什么需要受控源？

先问你一个问题：**三极管（晶体管）怎么用电路模型描述？**

三极管的工作原理：基极小电流 $i_B$ 控制集电极大电流 $i_C = \beta i_B$。

这个关系用电阻能描述吗？❌ 不行，因为 $i_C$ 不依赖于集电极两端的电压，而依赖于**另一条支路**的电流。

用独立源能描述吗？❌ 也不行，因为独立源的值是固定的，而 $i_C$ 会随 $i_B$ 变化。

**所以我们需要一种新元件**：它的输出值受电路中其他某个电压/电流控制——这就是**受控源**。

---

#### 四种受控源

根据"控制量"和"被控量"的组合，共有四种：

|名称|简写|被控量|控制量|方程|系数单位|
|---|---|---|---|---|---|
|压控电压源|**VCVS**|电压 $u_2$|电压 $u_1$|$u_2 = \mu u_1$|无量纲|
|压控电流源|**VCCS**|电流 $i_2$|电压 $u_1$|$i_2 = g u_1$|西门子 S|
|流控电压源|**CCVS**|电压 $u_2$|电流 $i_1$|$u_2 = r i_1$|欧姆 Ω|
|流控电流源|**CCCS**|电流 $i_2$|电流 $i_1$|$i_2 = \beta i_1$|无量纲|

**符号**：受控源用**菱形**表示，和独立源（圆形）区分开。

```
  独立电压源       受控电压源
     ○               ◇
    (+)             (+)
```

---

#### 受控源的本质：它不是电源，是"约束关系"

这是理解受控源最关键的一句话，请你好好咂摸这句话。

**普通电源**（独立源）：主动提供能量，它的值由"自己"决定。

**受控源**：它的值**完全由控制量决定**，控制量为零它就为零。它更像是一种"**依赖关系**"或"**传递函数**"，描述的是电路某一部分如何"反映"另一部分的状态。

#### 一个直观的例子

```
    i1 →                    i2 = 3·i1
  ────────┐              ┌──────────
          │              │
         [1Ω]           [◇]← CCCS
          │              │
          └──────┬───────┘
                 │
                 地
```

左边支路电流 $i_1$ 控制右边 CCCS 输出 $i_2 = 3i_1$。

- 如果左边 $i_1 = 2\text{A}$，那右边 $i_2 = 6\text{A}$
- 如果左边 $i_1 = 0$，那右边 $i_2 = 0$——**受控源自己不会"发电"**

---

#### 受控源的 KCL/KVL 处理

**好消息**：KCL、KVL 对受控源**完全适用**，把它当普通电源列方程就行。

**难点在哪里？**：方程里会多出一个"控制量"变量，你需要**再补一个约束方程**把它和电路里的实际量挂钩。

看个小例子：

```
   10V ──[2Ω]──┬──[3Ω]── 地
        I→    │
              ◇  ← CCVS: U = 4·I (电压参考方向 +在上，−在下)
              │
              地
```

设电流 I（流过 2Ω 的电流）向右。列节点 KCL 和 KVL 时：

- 10V 支路、2Ω、3Ω 支路正常处理
- CCVS 作为一个电压为 $4I$ 的"电压源"处理
- **额外约束**：CCVS 的电压值 = 4 × (2Ω 支路的电流 I)

这个"额外约束"就是受控源带来的新东西。**两个未知量（I 和 CCVS 电压），两个方程（KCL/KVL + 约束）**，就能解出。

## 第二章 线性直流电路（系统分析方法）

第一章我们学了"基本语言"（KCL、KVL、元件 VCR）。但当电路变复杂时，**节点多、回路多、方程一大堆**，怎么办？

第二章就是解决这个问题：**如何系统化、程式化地列出最少的独立方程，高效求解任意电路。**

---

### 2.1 网络的图（Graph Theory 基础）

#### 为什么要学"图"？

你列出的难点之一是"**独立变量和独立方程**"。这个难点的根源就在这里：

- 一个电路有多少节点？多少回路？
- 能列多少个 KCL？多少个 KVL？
- 其中有多少个是**独立**的（不重复、不冗余）？

要严谨回答这些问题，就需要借助**图论**。

#### 电路 → 图

把电路中每个**二端元件**（不管是电阻、电容还是电源）都**抽象成一条线段**（叫**支路 branch**），连接两个端点（叫**节点 node**），就得到了电路的"图"。

```
   电路：                    图：
   
   ┌──[R1]──┬──[R2]──┐      ┌────┬────┐
   │        │        │      │    │    │
   U        [R3]     I      │    │    │  
   │        │        │      │    │    │
   └────────┴────────┘      └────┴────┘
```

- 图**只保留连接关系**，不关心元件是什么
- 如果支路标了方向（参考方向），就是**有向图**

#### 三个关键术语

**① 支路（branch, b）**：每条线段，对应一个二端元件

**② 节点（node, n）**：支路的端点

**③ 回路（loop）**：从某节点出发，沿支路走一圈回到起点，途中不重复经过节点

#### 树（Tree）—— 这是核心概念

**树的定义**：

> 从图中选出一组支路，满足：
> 
> 1. **连接所有节点**
> 2. **不形成任何回路**
> 
> 这组支路就构成一棵"树"，其中的支路叫**树支（tree branch）**，剩下没选的支路叫**连支（link）**。

#### 一个例子

```
图的结构（4 个节点 a,b,c,d，5 条支路 1,2,3,4,5）：

     a ──1── b
     │       │
     2       3
     │       │
     c ──4── d
         │
         5
         (5 是 a-d 之间的对角线支路)
```

一种可能的树选法：选 {1, 2, 3} 作树支

```
   树支（实线）：  a ──1── b
                  │       │
                  2       3
                  │       │
                  c       d
   
   连支（虚线）：  c ──4── d, a ----5---- d
```

**验证**：树支 {1,2,3} 连接了所有 4 个节点，且不形成回路。✓

#### 关键性质（记住这三条）

对于有 $n$ 个节点、$b$ 条支路的连通图：

|量|个数|含义|
|---|---|---|
|树支数|$n-1$|一棵树恰好有 $n-1$ 条树支|
|连支数|$b-(n-1) = b-n+1$|剩下的支路都是连支|
|独立回路数|$b-n+1$|**等于连支数**！|

**为什么独立回路数 = 连支数？**

> 每添加一条连支到树上，就**恰好形成一个新回路**（叫**基本回路**）。有几条连支，就有几个基本回路。这些基本回路相互独立，数目就是 $b-n+1$。

#### 这个公式解决了什么？

前面例子中的电路：$n=4$, $b=5$，独立回路数 = $5-4+1 = 2$。

这告诉我们：**最多只能列 2 个独立的 KVL 方程**。如果你写了 3 个 KVL，其中一个一定是另外两个的线性组合（废话）。

---

### 2.2 KCL 和 KVL 的独立方程

现在可以回答"独立方程"的问题了：

#### KCL 独立方程数 = $n-1$

**为什么？**

- 每个节点能列 1 个 KCL，共 n 个
- 但所有节点 KCL 相加：每条支路电流在一端"流入"、另一端"流出"，相加=0
- 即 n 个 KCL 加起来恒等于 0，说明它们**线性相关**
- 所以独立的只有 $n-1$ 个

**实操**：任选 $n-1$ 个节点列 KCL 即可，剩下那个节点（通常选作"参考节点"或"地"）不用列。

#### KVL 独立方程数 = $b-n+1$

就是上面说的独立回路数。**实操**：找出所有基本回路（每个连支对应一个），每个列一个 KVL。

#### 总结：独立方程总数

$\underbrace{(n-1)}_{\text{KCL}} + \underbrace{(b-n+1)}_{\text{KVL}} = b$

**正好等于支路数！** 这就对了：b 条支路有 b 个未知电流（或 b 个未知电压），需要 b 个独立方程才能解出。

---

### 2.3 支路电流法

#### 基本思路

最直接的方法：**以 b 个支路电流为未知数，列 b 个独立方程。**

步骤：

1. 标出所有支路电流的参考方向（共 b 个未知量）
2. 对 $n-1$ 个独立节点列 KCL
3. 对 $b-n+1$ 个独立回路列 KVL（电压用 RiRi Ri 表示）
4. 联立求解

#### 例题

```
       I1 →    I3 ↓    I2 ←
   ┌──[2Ω]──┬──[3Ω]──┬──[1Ω]──┐
   │        │         │        │
  (+)      [4Ω]      [5Ω]     (+)
  10V       │I4↓      │I5↓    5V
  (-)       │         │       (-)
   │        │         │        │
   └────────┴─────────┴────────┘
```

**分析**：

- 节点数 $n=4$（上面三个节点 + 下面一个地），独立 KCL：3 个
- 支路数 $b=5$（R1=2Ω, R2=1Ω, R3=3Ω, R4=4Ω, R5=5Ω... 等等，加上电源支路其实是 7 条）

- 未知数多（b 个）
- 方程多（b 个）
- 电路稍复杂就"炸"了

所以有了下面两种**更高效**的方法：回路电流法 和 节点电压法。

### 2.4 回路电流法

#### 核心思想

> **不直接求每条支路的电流，而是求每个"独立回路"里的"假想循环电流"。**

想象每个独立回路里有一个**绕圈流动的电流**（叫**回路电流**或**网孔电流**），这样：

- **未知数从 b 个减少到 $b-n+1$ 个**（等于独立回路数）
- 每个回路电流**自动满足 KCL**（转一圈流入=流出）
- **只需要列 KVL** 方程

#### 关键点：支路电流 = 相关回路电流的代数和

一条支路可能只属于一个回路，也可能是两个回路的**共有支路**。

```
    回路 I_a          回路 I_b
    ┌─────┐          ┌─────┐
    │     │ 共有 R   │     │
    │     ├──[R]──────┤     │
    │     │←I_a  I_b→│     │
    └─────┘          └─────┘
```

共有电阻 R 上的实际电流 = $I_a - I_b$（两回路电流方向相反时相减；相同时相加）。

#### 标准步骤

1. 选一棵树，每个连支对应一个独立回路（实操中常选**网孔**——平面电路中最小的回路）
2. 设每个回路电流，标**绕行方向**（通常都取顺时针，便于统一）
3. 对每个回路列 KVL
4. 解方程

#### 标准方程形式（自阻 & 互阻）

对于回路 k：

$R_{kk} I_k - \sum_{j \neq k} R_{kj} I_j = \sum U_{sk}$

- $R_{kk}$：**自阻** = 回路 k 中所有电阻之和（恒为正）
- $R_{kj}$：**互阻** = 回路 k 和 j 共有的电阻（若两回路电流在共有支路上方向相同则取正，相反则取负；如果所有回路都按同一方向绕，互阻前的符号就都是负）
- $\sum U_{sk}$：回路 k 中所有电压源的代数和（与绕行方向相同的电位"升"取正）

#### 例题：用回路法重做上面那道

```
       I1 →         I2 →
   ┌──[2Ω]──── a ───[3Ω]──┐
   │           │           │
  (+)        [6Ω]         (+)
  12V          │          6V
  (-)          │          (-)
   │           │           │
   └───────────┴───────────┘
```

设两个回路电流都**顺时针**：

- 左回路：$I_a$（经 12V、2Ω、6Ω）
- 右回路：$I_b$（经 6Ω、3Ω、6V）

**左回路 KVL**：

- 自阻：2+6=82 + 6 = 8 2+6=8 Ω
- 共有电阻（6Ω），$I_a$ 和 $I_b$ 方向相反，互阻项取负号
- 电压源：12V（顺时针绕经过，从"−"到"+"）

$8 I_a - 6 I_b = 12$

**右回路 KVL**：

- 自阻：6+3=96 + 3 = 9 6+3=9 Ω
- 互阻 −6Ω
- 电压源：−6V（顺时针绕经 6V 时从"+"到"−"）

$-6 I_a + 9 I_b = -6$

**解**：

$\begin{cases}
8I_a - 6I_b = 12 \\
-6I_a + 9I_b = -6
\end{cases}$

解得 $I_a = 2$ A，$I_b = \frac{2}{3}$ A

**支路电流**：

- $I_1 = I_a = 2$ A ✓
- $I_2 = -I_b = -\frac{2}{3}$ A（因为 $I_2$ 方向和 $I_b$ 相反）✓
- $I_3$（6Ω 向下）$= I_a - I_b = 2 - \frac{2}{3} = \frac{4}{3}$ A ✓

和支路电流法结果一致！但**只用了 2 个方程**而不是 3 个。

#### 💡 回路电流法的"陷阱"

**当电路中含有电流源时**，电流源两端电压未知，KVL 不好直接列。处理方法：

**情况一：电流源只属于一个回路** → 该回路的回路电流直接 = 电流源电流（已知），省一个方程

**情况二：电流源是两个回路的共有支路** → 引入电流源两端的**未知电压 U**，按电压源处理，补一个约束方程（两回路电流差 = 电流源值）。这种处理叫"**超网孔**"（supermesh）。

### 2.5 节点电压法（最常用！）

#### 核心思想

> **选一个节点作"地"（参考节点），其余节点对地的电压就是"节点电压"。以节点电压为未知数。**

- 未知数数量 = $n-1$（和 KCL 独立方程数一致）
- 每条支路电流可用**两端节点电压之差**除以电阻表示
- **只需要列 KCL** 方程

#### 为什么节点法用得最多？

- 未知数通常比回路法还少（对**节点少、回路多**的电路尤其高效）
- 所有电路仿真软件（如 SPICE）用的都是**改进节点法**
- 和电路拓扑关系直接，不用找回路

#### 标准方程形式（自导 & 互导）

对节点 k（取地为参考）：

$G_{kk} U_k - \sum_{j \neq k} G_{kj} U_j = \sum I_{sk}$

- $U_k$：节点 k 对地的电压
- $G_{kk}$：**自导** = 连接到节点 k 的所有电导之和（恒为正）
- $G_{kj}$：**互导** = 节点 k 和 j 之间支路的电导之和（前面有负号，所以方程里写 $-G_{kj} U_j$）
- $\sum I_{sk}$：流入节点 k 的电流源电流代数和（流入为正）

#### 例题：用节点法再做一遍

```
       I1 →         I2 →
   ┌──[2Ω]──── a ───[3Ω]──┐
   │           │           │
  (+)        [6Ω]         (+)
  12V          │          6V
  (-)          │          (-)
   │           │           │
   └───────────┴───────────┘
              地
```

选下方为地，只有一个未知节点电压 $U_a$。

**KCL @ 节点 a**（流入 = 流出）：

- 经 2Ω 从 12V 流入 a 的电流：$\dfrac{12 - U_a}{2}$
- 经 3Ω 从 6V 流入 a 的电流：$\dfrac{6 - U_a}{3}$
- 经 6Ω 从 a 流出（流向地）的电流：$\dfrac{U_a}{6}$

$\frac{12 - U_a}{2} + \frac{6 - U_a}{3} = \frac{U_a}{6}$

乘以 6：

$3(12-U_a) + 2(6-U_a) = U_a$ $36 - 3U_a + 12 - 2U_a = U_a$ $48 = 6U_a \implies U_a = 8 \text{ V}$

**支路电流**：

-  = (12-8)/2 = 2$ A ✓
- $I_2 = (6-8)/3 = -\frac{2}{3}\,\text{A}$ ✓
- $I_3 = 8/6 = \frac{4}{3}\,\text{A}$ ✓

🎉 **只用了 1 个方程**就解决了！这就是节点电压法的威力。

#### 节点法"陷阱"：含电压源时的处理

**情况一：电压源一端接地** → 另一端的节点电压直接 = 电压源电压（已知），省一个方程

**情况二：电压源连在两个非地节点间（"悬浮"电压源）** → 引入电压源电流作未知数（或用"**超节点**" supernode 技巧：把电压源两端的节点合并成一个"大节点"列 KCL，再补一个约束：两节点电压差 = 电压源电压）

### 2.6 含运算放大器电路的分析

#### 运放（Op-amp）是什么？

运放是个**五端元件**（实际上看作三端：两个输入 + 一个输出，电源端通常略去不画），符号：

```
      V+ ───▶│＼
              │  ＼___ Vo
      V− ───▶│  ／
              │／
```

**理想运放的两个黄金法则**（这是本节精髓）：

#### 黄金法则一："虚断"

> 两个输入端**电流为零**（输入阻抗无穷大）

$i_+ = i_- = 0$

#### 黄金法则二："虚短"

> 两个输入端**电位相等**（开环增益无穷大 + 负反馈）

$u_+ = u_-$

> ⚠️ **前提**：必须有**负反馈**，即输出通过某种方式回到反相输入端（V−）。没有负反馈时虚短不成立！

#### 经典例题：反相放大器

```
       R_f
    ┌──[Rf]──┐
    │        │
   v_in      │
    │        │       V−  ___
    └──[R1]──┴───────▶│＼
                       │  ＼___ V_out
               地─────▶│  ／
                       │／
                       V+
```

**分析**：

- V+ 接地 → $u_+ = 0$
- 虚短：$u_- = u_+ = 0$（叫“虚地”）
- 虚断：反相端无电流流入运放
- 所以 R1 的电流 = Rf 的电流（KCL）： $\frac{v_{in} - 0}{R_1} = \frac{0 - v_{out}}{R_f}$

$\boxed{v_{out} = -\frac{R_f}{R_1} v_{in}}$

输出与输入**反相**，放大倍数为 $R_f/R_1$。

#### 虚短 + 虚断的本质

其实运放可以看作一个**特殊的受控源**（VCVS，增益无穷大）。虚短虚断是"增益 → ∞"和"输入阻抗 → ∞"两个极限的数学结果。

### 📋 第二章小结：方法对比

|方法|未知数个数|方程类型|适用场景|
|---|---|---|---|
|支路电流法|b（支路数）|KCL + KVL|理论基础，实际少用|
|**回路电流法**|b-n+1|KVL|回路少、支路多时|
|**节点电压法**|n-1|KCL|节点少、支路多时（**最常用**）|

**选择原则**：看电路中**哪个数更少**，就用哪种方法。

### 3.1 置换定理（Substitution Theorem）

#### 先用一个问题引入

假设我解出了某电路的一条支路：电流  = 2$ A，电压  = 6$ V。

现在我把这条支路"挖掉"，换成一个 **2A 的电流源**，电路其他部分会变吗？

---

**稍停一下，请你自己想想**：

换上 2A 电流源后，原先连接这条支路的两个节点之间，电流还是 2A 吗？电压还是 6V 吗？电路其他部分的电流电压会变吗？

（凭直觉答，不用严谨证明）

---

#### 答案：不会变！

**置换定理**的内容：

> 电路中任一支路 $k$，若已知其电压 $U_k$ 和电流 $I_k$，则可以用以下任一“等效元件”置换它，电路其他部分的**所有电压电流保持不变**：
> 
> - 一个电压为 $U_k$ 的**电压源**
> - 一个电流为 $I_k$ 的**电流源**
> - 一个电阻 $R = U_k / I_k$（只要 $I_k \neq 0$）

#### 为什么？（直觉理解）

电路其他部分“看到的”这条支路，只是一个“两端之间有电压 $U_k$、流过电流 $I_k$”的黑箱。**黑箱里是什么不重要**，只要端口上的 $U, I$ 一致，外部感受不到任何差别。

这就像你问"隔壁屋子里有人在说话"——你只听到声音，里面是录音机还是真人，你听不出区别。

#### 一个用法示例

```
   ┌──[R1]──┬──[R2]──┐
   │        │        │
  (+)      [R3]     [R4]
   U        │        │
  (-)       │        │
   │        │        │
   └────────┴────────┘
```

假设通过某种方法我算出了 R3 上电流  = 1$ A。

**接下来想求 R4 上电压？**

- 把 R3 换成 **1A 电流源**——R3 上的电阻值对后续计算毫无影响！
- 电路变简单了：现在是"一个电压源 U，串联 R1，并联一个 1A 电流源（方向固定），再并联一个 R2-R4 支路"

这招在分析时非常省事——**已知量直接"冻结"**，让剩下的部分聚焦在未知量上。

### 3.2 叠加原理（Superposition）

#### 核心思想（一句话）

> **在线性电路中，多个独立源共同作用产生的响应，等于每个独立源单独作用时产生的响应之和。**

注意三个关键词：

1. **线性电路**（所有元件都是线性的）
2. **独立源**（受控源不参与叠加，始终保留）
3. **响应**（可以是某支路电流，也可以是某元件电压）

#### "单独作用" 是什么意思？

**让一个源单独作用** = **把其他独立源"置零"**：

- 电压源置零 = **短路**（因为电压源 U=0 就是一根导线）
- 电流源置零 = **开路**（因为电流源 I=0 就是断开）

> ⚠️ **注意**：只置零**独立源**！**受控源必须保留**（因为它不是"源"，是一种约束关系）。

#### 一个例子

```
   ┌──[2Ω]──── a ───[3Ω]──┐
   │           │           │
  (+)        [6Ω]         ↑
  12V          │          2A
  (-)          │          ↓
   │           │           │
   └───────────┴───────────┘
              地
```

求节点 a 的电压 $U_a$。

**第一步：12V 单独作用**（2A 电流源开路）

```
   ┌──[2Ω]──── a ───[3Ω]──┐
   │           │           │
  (+)        [6Ω]      开路（悬空）
  12V          │        
  (-)          │           
   │           │           
   └───────────┴──────────
```

3Ω 右端开路，故 3Ω 上无电流、无压降。所以节点 a 的电压就等于 6Ω 上的电压。

这是一个简单的分压：12V 电压源经 2Ω 接到 a，a 经 6Ω 接地。

$U_a^{(1)} = 12 \times \frac{6}{2+6} = 9\,\text{V}$

**第二步：2A 电流源单独作用**（12V 电压源短路）

```
   ┌──[2Ω]──── a ───[3Ω]──┐
   │           │           │
  短路      [6Ω]          ↑
   │           │          2A
   │           │          ↓
   └───────────┴───────────┘
```

12V 短路后，2Ω 左端直接接地。所以从 a 往左看是 2Ω，往下看是 6Ω，这两个**并联**：

$R_{\text{左下}} = 2 \parallel 6 = \frac{2 \times 6}{2+6} = 1.5\,\Omega$

右边 2A 电流源经过 3Ω 到 a。节点 a 对地的等效电阻路径：往右看 3Ω，然后...

等等，让我重新梳理一下。2A 电流源（假设箭头向上）接在 3Ω 右端和地之间。那么 2A 电流从地流出，经电流源向上，再经 3Ω 从右向左流入节点 a，然后从 a 分流到 2Ω 和 6Ω 去。

节点 a 看向地的等效电阻：2Ω ∥ 6Ω = 1.5Ω

流入 a 的电流为 2A（经 3Ω 进来），则：

$U_a^{(2)} = 2 \times 1.5 = 3\,\text{V}$

**叠加**：

$U_a = U_a^{(1)} + U_a^{(2)} = 9 + 3 = 12\,\text{V}$

#### 齐性定理（顺带提一下）

**齐性**（Homogeneity）：在线性电路中，**所有独立源同时放大 k 倍，所有响应也放大 k 倍**。

叠加 + 齐性 = **线性性质**的两个方面。

#### 叠加原理的典型陷阱

**❌ 功率不能叠加！**

$P = UI$ 是乘积，不是线性关系。

比如：电阻 $R$ 上，源 1 单独作用时电压、电流分别为 $U_1, I_1$，源 2 单独作用时分别为 $U_2, I_2$：

$P_{\text{总}} = (U_1+U_2)(I_1+I_2) \neq U_1 I_1 + U_2 I_2$

**求功率必须先叠加出总电压/总电流，再算功率！**

### 3.3 等效电源定理（本章最重要！⭐）

这是电路分析中**使用频率最高**的定理，你要熟到"看电路就条件反射"的程度。

#### 3.3.1 戴维南定理（Thévenin's Theorem）

##### 定理内容

> 任意一个**线性含源二端网络**（从外部看是两个端子的电路），对外电路而言，都等效于一个**电压源串联电阻**的组合：
> 
> - 电压源值 $U_{oc}$ = 该二端网络**开路电压**
> - 串联电阻 $R_{th}$ = 该二端网络内部所有独立源置零后的**等效电阻**

```
  复杂线性网络        戴维南等效
     ┌───┐               ┌──[R_th]──┐
     │ ? │ a              │          │ a
     │   │    ═══►      (+)         
     │ ? │ b             U_oc        
     └───┘               (-)           
                          │          │ b
                          └──────────┘
```

##### 为什么这么有用？

想象你在设计电路：电源部分可能非常复杂（几十个元件），但你关心的"负载"只要通过**两个端子**连接上去。

→ **把电源部分打包成一个"U_oc 串 R_th"的黑箱**，负载想换什么就换什么，计算超级简单。

##### 求戴维南等效的三步法

**步骤 1：求开路电压 $U_{oc}$**

把外电路断开（端口开路），用任何方法（节点法、回路法、叠加法……）求端口电压。

**步骤 2：求等效电阻 $R_{th}$**

这一步有**三种方法**，按情况选择：

**方法 A：置零法**（电路中**只有独立源**时）

- 所有独立源置零（电压源短路、电流源开路）
- 从端口看进去的电阻即为 $R_{th}$

**方法 B：外加电源法**（含**受控源**时⚠️）

- 独立源置零，但受控源保留
- 在端口外加一个测试电压 $U_T$（或电流 $I_T$）
- 求端口响应电流 $I_T$（或电压 $U_T$）
- $R_{th} = U_T / I_T$

**方法 C：开路短路法**

- 求开路电压 $U_{oc}$
- 求**端口短路**时的短路电流 $I_{sc}$
- $R_{th} = U_{oc} / I_{sc}$

**步骤 3：画出等效电路**

```
      ┌──[R_th]──┐
      │          │
     (+)        
     U_oc       端口 a
     (-)           端口 b
      │          │
      └──────────┘
```

---

#### 3.3.2 诺顿定理（Norton's Theorem）

##### 定理内容

> 任意线性含源二端网络，对外电路而言也等效于一个**电流源并联电阻**的组合：
> 
> - 电流源 $I_{sc}$ = 该网络**端口短路**时的短路电流
> - 并联电阻 $R_N$ 和戴维南电阻相同（即 $R_N = R_{th}$）

```
     ┌──────────┐
     │          │
     ↑        [R_N]   
    I_sc        │    端口 a
     ↓          │    
     │          │    端口 b
     └──────────┘
```

##### 戴维南 ↔ 诺顿 互相转换

这两种形式可以**互相转换**（电源等效变换）：

$I_{sc} = \frac{U_{oc}}{R_{th}}, \quad R_N = R_{th}$

```
  戴维南                    诺顿
  ┌──[R]──┐               ┌───────┐
  │       │               │       │
 (+)      │    ═══►       ↑      [R]
  U       │               U/R     │
 (-)      │               ↓       │
  │       │               │       │
  └───────┘               └───────┘
```

**实用技巧**：求等效电路时，哪种好求就先求哪种，需要另一种时转换一下。

---

#### 经典例题：戴维南等效

```
   ┌──[2Ω]──┬──[3Ω]──┐ a
   │        │        │
  (+)      [6Ω]      │
  12V       │        │ 
  (-)       │        │ b
   │        │        │
   └────────┴────────┘
```

求 a-b 端口的戴维南等效电路。

**步骤 1：求 $U_{oc}$**（端口 a-b 开路）

a-b 开路意味着 3Ω 右端悬空，故 3Ω 上无电流、无压降。则：

$U_{oc} = U_{ab} = U_{6\Omega} = 12 \times \frac{6}{2+6} = 9\,\text{V}$

**步骤 2：求 $R_{th}$**（12V 电压源短路）

```
   ┌──[2Ω]──┬──[3Ω]──┐ a
   │        │        │
  短路    [6Ω]      │
   │        │        │
   │        │        │ b
   └────────┴────────┘
```

从 a-b 往里看：经过 3Ω，到节点 ×，往下是 6Ω、往左是 2Ω（左端接地 = 接 b 端）。

所以 2Ω 和 6Ω **并联**，再和 3Ω **串联**：

$R_{th} = 3 + (2 \parallel 6) = 3 + 1.5 = 4.5\,\Omega$

**最终戴维南等效**：

```
     ┌──[4.5Ω]──┐
     │          │
    (+)         │ a
    9V         
    (-)         │ b
     │          │
     └──────────┘
```

如果现在把一个负载 $R_L = 4.5\,\Omega$ 接到 a-b 端口，负载上电流是多少？  
直接 $I = \dfrac{9}{4.5+4.5} = 1\,\text{A}$，不用再去算原始电路！

#### 含受控源时的戴维南求法（难点⚠️）

**例题**：

```
   ┌──[2Ω]──┬──────── a
   │        │    
  (+)       │    ◇ ← CCVS: U = 3i
  12V       │    │   (i 是 2Ω 上的电流，向右为正)
  (-)       │    │    
   │        │    │ b
   └────────┴────┘
```

受控源 CCVS 在 2Ω 右端和 b 端之间，电压 $U_{CCVS} = 3i$（+上 −下），其中 $i$ 是流过 2Ω 的电流（向右）。

**求 a-b 戴维南等效**。

---

**① 求 $U_{oc}$**（a-b 开路）

a-b 开路 → i = 0（2Ω 上电流为 0，因为回路断开）

→ CCVS 电压 = 3×0 = 0

→ $U_{oc} = 12\,\text{V}$（12V 电源直接通过 2Ω 和 CCVS 到 a，但 2Ω 无压降，CCVS 也为 0）

等等，这里要小心。a-b 开路，但 12V 电源还在。12V 电源的电流能不能流？回路是：12V → 2Ω → CCVS → b → 回到 12V 负极。这个回路是闭合的！

让我重新画：

```
   ┌──[2Ω]──────── a
   │         
  (+)        
  12V       ◇ CCVS
  (-)       U=3i
   │         │
   └─────────┴──── b
```

a-b 开路时，外部没有回路。但内部呢？i 流过 2Ω 后必须有地方去。如果电路只有左边 12V-2Ω-CCVS-回到 12V 这一个回路，那么 i 在这个闭合回路中流动。

**KVL（左回路）**：$12 - 2i - 3i = 0 \implies i = 2.4\,\text{A}$。  
**但等等**，CCVS 在 a 和 b 之间，a-b 开路意味着 a 是悬空的...

我画的电路不够清楚，让我**重新设计一个更明确的例题**：

```
           i →
   ┌──[2Ω]──────┬────── a
   │            │
  (+)          ◇ ← CCVS: U_CCVS = 4i
  10V          │   (电压参考 +上 −下)
  (-)          │   
   │           │
   └───────────┴────── b
```

**CCVS 串联在 a-b 主回路的"下支路"里**，即：电源→2Ω→节点×→(a)，节点×→CCVS→(b)。

等下，这样 CCVS 和 a 直接连了，结构还是不清晰。

**例题（含受控源的戴维南）**：

电路中有一个 10V 独立电压源、一个 2Ω 电阻 $R_1$、一个 3Ω 电阻 $R_2$，和一个 CCVS（流控电压源，$U_{\text{控}} = 2i_1$，其中 $i_1$ 是流过 $R_1$ 的电流）。

连接关系：10V 电源的 + 端经 $R_1$（电流 $i_1$ 向右为正）到节点 a；节点 a 经 $R_2$ 到 CCVS 的 + 端；CCVS 的 − 端接地；10V 的 − 端也接地。求 a-**地**端口的戴维南等效。

**这里 "端口" 就是节点 a 和地之间。**

**① 求 $U_{oc}$**

a 端开路意味着 $R_2$ 右端悬空，故 $R_2$ 支路电流为 0，$R_2$ 上无压降。

但 $R_1$ 支路是闭合回路吗？$R_1$ 右端接 a，a 开路，所以 $R_1$ 也没有电流了！$i_1 = 0$。

$\Rightarrow U_{CCVS} = 2 \times 0 = 0$

$\Rightarrow U_{oc} =$ 节点 a 的电位 $= 10\,\text{V}$（因为 $R_1$ 无压降，a 直接等于 10V 电源的正极电位）

$U_{oc} = 10\,\text{V}$

**② 求 $R_{th}$**（独立源置零，10V 短路；**受控源保留**）

10V 短路后：$R_1$ 左端接地。外加一个测试电压 $U_T$ 到 a-地端口。

- 设从端口流入的测试电流为 $I_T$
- 此时 $i_1$ 是从地经 $R_1$ 流向 a 的电流...

等等，现在 a 接外部 $U_T$，不是悬空了。$R_1$ 从地（原 10V 负极）到 a，如果 $U_a = U_T$，则设 $i_1$ 向右为正（从地流向 a），那么

$i_1 = \dfrac{0 - U_T}{R_1} = -\dfrac{U_T}{2}$

$U_{CCVS} = 2i_1 = -U_T$

应用 KVL 分析 $R_2$-CCVS 支路：从 a 经 $R_2$ 到 CCVS 的 + 端，CCVS 的 − 端接地。

$U_T - I_{R_2} \cdot R_2 - U_{CCVS} = 0$（沿这条路径 KVL，到地）

$U_T - I_{R_2} \cdot 3 - (-U_T) = 0$

$2U_T = 3I_{R_2}$

$I_{R_2} = \frac{2U_T}{3}$（从 a 流向地）

**测试电流 $I_T$** = 流入端口的总电流 = $I_{R_2}$ + （流向 $R_1$ 方向的电流）

流向 $R_1$ 方向的电流 $= -i_1 = U_T/2$（因为 $i_1$ 定义为从地流向 a，而测试电流是从 a 流向地）

$I_T = \frac{2U_T}{3} + \frac{U_T}{2} = \frac{4U_T + 3U_T}{6} = \frac{7U_T}{6}$

$R_{th} = \frac{U_T}{I_T} = \frac{6}{7}\,\Omega$

**戴维南等效**：$10\,\text{V}$ 串联 $\frac{6}{7}\,\Omega$。

---

我知道这个例子的推导有点冗长。**关键思路你必须记住**：

> **含受控源时，求 $R_{th}$ 用“外加电源法”**：独立源置零，受控源保留，外加测试源，算 $U_T/I_T$。

做完回复答案，我们就能顺畅进入 **3.4 特勒根定理** 和 **3.5 互易定理**。

## 3.4 特勒根定理（Tellegen's Theorem）

这个定理是个"怪才"——它听起来平平无奇，但威力巨大。

### 定理内容

> 对**任意电路**，只要所有支路电压 $u_k$ 取“关联参考方向”下的电压，所有支路电流 $i_k$ 取同样的参考方向，则：
> 
> $\sum_{k=1}^{b} u_k i_k = 0$

这个等式说：**全电路所有支路上 $u_k i_k$ 的代数和恒等于零**。

### 它的物理意义是什么？

关联参考方向下，$u_k i_k$ 就是该支路**吸收**的功率。所以这个公式说的是：

> **整个电路"吸收的总功率" = 0**

换句话说——电源释放的功率 = 所有元件吸收的功率。这就是**能量守恒**！

### 特勒根定理厉害在哪里？

关键是**它对"任意电路"成立**——包括非线性电路、时变电路。只要满足 KCL 和 KVL 就行。

更奇妙的是**特勒根定理的推广形式**：

> 有两个电路 $N$ 和 $\hat{N}$，**拓扑结构完全相同**（图相同），但元件可以完全不同。用 $u_k$ 表示 $N$ 的支路电压，用 $\hat{i}_k$ 表示 $\hat{N}$ 的支路电流（参考方向一致），则：
> 
> $\sum_{k=1}^{b} u_k \hat{i}_k = 0 \quad \text{和} \quad \sum_{k=1}^{b} \hat{u}_k i_k = 0$

两个**不同电路**的电压和电流"混搭"起来，求和也等于 0！这就神奇了。

### 直觉理解

为什么会这样？

- KCL 只和**电路的图**有关（谁连着谁）
- KVL 也只和**电路的图**有关
- 只要两个电路的图相同，N 的电压和 N^\hat{N} N^ 的电流就"恰好配得上"——像拼图一样能对上

### 用途

特勒根定理主要是**理论证明工具**：

- 证明互易定理（下一节）
- 证明网络的其他对称性
- 电路灵敏度分析

**作为一年级本科生，你只需要记住**：

1. 普通形式 → 能量守恒
2. 推广形式 → 同拓扑电路电压电流的"奇妙乘积关系"
3. 这是一个"理论桥梁"，算具体数值题不常用

## 3.5 互易定理（Reciprocity Theorem）

### 先看一个神奇的现象

```
           电路 N₁（原电路）                     电路 N₂（互易后）
   ┌──[R1]──┬──[R3]──┐                 ┌──[R1]──┬──[R3]──┐
   │        │        │                 │        │        │
  (+)      [R2]     (I₂?)              │      [R2]      (+)
   U        │        │                (I₁?)     │        U
  (-)       │        │                 │        │        │
   │        │        │                 │        │        │
   └────────┴────────┘                 └────────┴────────┘
     ①左端放电源               ②把电源搬到右端（同样的U），
     测右端电流 I₂                测左端电流 I₁
```

**结论**：对于**线性无独立源、无受控源**的电路， = I_2$。

**激励（电源）和响应（电流）位置互换，数值不变**！这就是互易定理。

### 三种形式

**形式一：电压源激励 → 电流响应**

在 1-1' 端加电压源 $U_s$，在 2-2' 端**短路**测电流 $I_2$。

把电源移到 2-2' 端（同样 $U_s$），在 1-1' 端**短路**测电流 $I_1$。

结论：$\boxed{I_1 = I_2}$

**形式二：电流源激励 → 电压响应**

在 1-1' 端加电流源 $I_s$，在 2-2' 端**开路**测电压 $U_2$。

把电流源移到 2-2' 端（同样 $I_s$），在 1-1' 端**开路**测电压 $U_1$。

结论：$\boxed{U_1 = U_2}$

**形式三：电压源激励 → 电压响应（注意！要求比值）**

在 1-1' 端加电压源 $U_s$，在 2-2' 端**开路**测电压 $U_2$。

把电压源**换成电流源** $I_s$，移到 2-2' 端，在 1-1' 端**开路**测电压 $U_1$。

结论：$\boxed{\dfrac{U_1}{I_s} = \dfrac{U_2}{U_s}}$

### 互易定理的前提（⚠️ 非常重要）

互易定理**不是对所有线性电路都成立**！必须满足：

1. **线性** ✓
2. **只含电阻**（或更一般地：**互易元件**，电容、电感、变压器也行）
3. **不含受控源**⚠️（受控源会破坏互易性）
4. **不含独立源**（激励源单独考虑，内部不能有其他独立源）

### 一个例子

```
   ┌──[2Ω]──┬──[3Ω]──┐
   │        │        │
  (+)      [6Ω]      短路
  10V       │        │
  (-)       │        │
   │        │        │
   └────────┴────────┘
```

左端 10V 激励，右端短路。求短路电流 II I。

**直接算**：

- 从 a-b 端口看进去（把 10V 短路）：2Ω 和 6Ω 并联，再和 3Ω 串联

等等，让我直接求：节点 a（上方中间）对地电压 $U_a$：

节点方程：$\dfrac{10 - U_a}{2} = \dfrac{U_a}{6} + \dfrac{U_a}{3}$

解得  = 5$ V。右端短路电流 =  / 3 = 5/3$ A。

**根据互易定理**：若把 10V 搬到右端（短路到地），2Ω 左端短路测电流，结果也应该是 5/3 A。（你有兴趣可以验算，会发现确实如此。）

### 互易定理的用途

- **双口网络分析**（第十一章会用到）
- **天线理论**（发射天线 = 接收天线，就是互易定理的应用）
- **电网络对称性的判定**

## 4.1 电容元件（大学升级版）

高中你学的电容，核心是 **“电荷储存器”**，满足 $Q = CU$，是静电学里的元件。

大学电路里，电容的"身份"升级了：**它是一个有时间动态的元件**。我们关心的不再是"储了多少电"，而是"电压电流随时间怎么变"。

---

### 一、电容的 VCR（伏安关系）—— 核心公式

从高中的 $Q = CU$ 出发，我们做一件简单的事：**对时间求导**。

$Q = Cu$

两边对 t 求导（C 是常数）：

$\frac{dQ}{dt} = C \frac{du}{dt}$

而 $\dfrac{dQ}{dt}$ 就是电流 $i$（单位时间流过的电荷量）：

$\boxed{i = C \frac{du}{dt}}$

**这就是电容的 VCR**。它是本章最重要的公式，你必须刻在脑子里。

---

### 二、这个公式揭示的五个惊人事实

#### 事实 ①：电容电流取决于电压的"变化率"，不是电压本身

这和电阻截然不同：

- 电阻： = u/R$ → 电压越大，电流越大
- 电容：$i = C\,du/dt$ → 电压变化**越快**，电流越大

**一个极端例子**：电容两端加一个 1000V 的**恒定**直流电压，电流是多少？

/dt = 0$（不变）→ i=0i = 0 i=0

**电容在直流稳态下等效于开路！** 这是重要结论。

#### 事实 ②：电容电压不能突变

假设电压突变（比如 0V 瞬间跳到 5V），那 $du/dt = \infty$，需要无穷大电流，物理上不可能。

$\boxed{\text{电容电压不能突变：} \; u_C(0^+) = u_C(0^-)}$

这条在第九章分析动态电路初始条件时是**金科玉律**。

#### 事实 ③：电容有"记忆"

把 VCR 反过来积分：

$u(t) = \frac{1}{C}\int_{-\infty}^{t} i(\tau)\,d\tau$

或者从某个初始时刻 $t_0$ 开始：

$u(t) = u(t_0) + \frac{1}{C}\int_{t_0}^{t} i(\tau)\,d\tau$

看到了吗？**t 时刻的电压，取决于过去所有时刻的电流**。这就是"记忆"——电容记得它经历过什么。

对比电阻 $u=Ri$：电阻是“无记忆”的，此刻电压只看此刻电流。

#### 事实 ④：电容储能

从功率入手，关联参考方向下：

$p = ui = u \cdot C\frac{du}{dt}$

从 −∞-\infty −∞（电压为 0）积分到当前时刻：

$W_C = \int p\,dt = \int_0^u Cu'\,du' = \frac{1}{2}Cu^2$

$\boxed{W_C = \frac{1}{2}Cu^2}$

**电容储存的是电场能量**（电介质中的电场）。注意：它**储能不耗能**——能存进去、能放出来，但不消耗（理想情况）。

#### 事实 ⑤：电容是"无源"但"动态"的元件

- **无源**：长期来看能量只能"过一下手"，不能无中生有
- **动态**：VCR 里有导数/积分，所以它是"动态元件"

对比总结：

|元件|VCR|动态/静态|耗能/储能|
|---|---|---|---|
|电阻 R|$u = Ri$|静态（代数关系）|耗能|
|电容 C|$i = C\,du/dt$|**动态**（微分关系）|**储能**（电场）|

---

### 三、电容的串并联

这部分规律和电阻**刚好相反**，记的时候要小心：

**并联**（电压相同，电流相加）：

$C_{\text{并}} = C_1 + C_2 + \cdots$

（像电阻串联的公式）

**串联**（电流相同，电压相加）：

$\frac{1}{C_{\text{串}}} = \frac{1}{C_1} + \frac{1}{C_2} + \cdots$

（像电阻并联的公式）

## 4.2 电感元件（全新内容，从零开始）

### 一、电感是什么？物理直觉先行

电感最常见的形式就是一个**线圈**：

```
          ┌─────────┐
     ─────┤ ∿∿∿∿∿∿ ├─────
          └─────────┘
```

**物理现象**：通电线圈会产生磁场（奥斯特实验，高中学过）。反过来，变化的磁场会在线圈里感应出电压（法拉第电磁感应定律）。

**关键一步——"自感应"**：

假设你在线圈里通电流 $i$，它产生磁场 $\Phi$。现在让 $i$ 变化 → 磁场 $\Phi$ 变化 → 线圈**自己**就会感应出一个电压来“反抗”这个变化。

$u = \frac{d\Phi}{dt}$

而磁通 $\Phi$ 与电流成正比：$\Phi = Li$（L 是电感系数，单位亨利 H）

两式合并：

$\boxed{u = L \frac{di}{dt}}$

**这就是电感的 VCR**。

---

### 二、电感 vs 电容——完美的"对偶"

把两个 VCR 对比着看：

||电容|电感|
|---|---|---|
|VCR|$i = C \dfrac{du}{dt}$|$u = L \dfrac{di}{dt}$|
|关心的变量|**电压**不能突变|**电流**不能突变|
|直流稳态|**开路**（电流=0）|**短路**（电压=0）|
|储能|$W_C = \dfrac{1}{2}Cu^2$（电场）|$W_L = \dfrac{1}{2}Li^2$（磁场）|
|串并联|和电阻**相反**|和电阻**相同**|

看出来了吗？**电感和电容在公式形式上完全对偶**——只要把"u"和"i"、"C"和"L"对换，两个元件的所有性质都能互换。

> 💡 **记忆技巧**：记住电容的所有性质，电感只要"对偶一下"就能推出来，不用单独记。

---

### 三、电感五个性质（对偶推出）

#### 性质 ①：电压取决于电流"变化率"

- 电流不变 → $di/dt = 0$ → $u=0$ → **直流稳态下电感等效于短路**

#### 性质 ②：电流不能突变

$\boxed{i_L(0^+) = i_L(0^-)}$

#### 性质 ③：电感有"记忆"

$i(t) = i(t_0) + \frac{1}{L}\int_{t_0}^{t} u(\tau)\,d\tau$

#### 性质 ④：储能公式

$W_L = \frac{1}{2}Li^2 \quad \text{（磁场能量）}$

#### 性质 ⑤：串并联规律（和电阻相同）

**串联**：$L_{\text{串}} = L_1 + L_2 + \cdots$

**并联**：$\dfrac{1}{L_{\text{并}}} = \dfrac{1}{L_1} + \dfrac{1}{L_2} + \cdots$

## 4.2 耦合电感（互感）

前面讲的电感是**单独一个线圈**的"自感应"。现在我们让两个线圈**靠近**，看会发生什么。

---

### 一、先用物理直觉打地基

#### 场景想象

两个线圈 L₁ 和 L₂ 靠得很近（比如绕在同一根铁芯上）：

```
      i₁ →                    i₂ →
   ┌──[L₁]──┐             ┌──[L₂]──┐
   │  ∿∿∿∿  │             │  ∿∿∿∿  │
   + u₁  -                  + u₂  -
               ╲ 磁场耦合 ╱
                ─────────
```

**Q（请你思考）**：如果我只在 L₁ 里通电流 $i_1$（L₂ 断开不通电），L₂ 两端会有电压吗？

---

**答**：**会有！**

为什么？$i_1$ 在 L₁ 中产生磁场，这个磁场**会穿过 L₂**（因为两个线圈靠得近）。如果 $i_1$ 是变化的，穿过 L₂ 的磁通就变化，根据法拉第电磁感应定律，L₂ 里就感应出电压。

**这就是"互感"**：一个线圈的电流变化在另一个线圈中感应电压的现象。

---

### 二、数学描述

#### 自感磁通 vs 互感磁通

现在 L₁ 里的**总磁通**来自两个源：

- **自己**的电流 $i_1$ 产生的磁通：$\Phi_{11} = L_1 i_1$（自感磁通）
- **L₂** 的电流 $i_2$ 穿过来的磁通：$\Phi_{12} = M i_2$（互感磁通，$M$ 是互感系数）

同样 L₂ 的总磁通：

- 自感磁通：$\Phi_{22} = L_2 i_2$
- 互感磁通：$\Phi_{21} = M i_1$

（理论上两个互感系数叫 $M_{12}$ 和 $M_{21}$，但可以证明 $M_{12} = M_{21} = M$，简化成一个常数。）

#### 端口电压

每个线圈两端的电压 = 总磁通对时间的导数：

$\boxed{u_1 = L_1 \frac{di_1}{dt} \pm M \frac{di_2}{dt}}$ $\boxed{u_2 = L_2 \frac{di_2}{dt} \pm M \frac{di_1}{dt}}$

注意那个 **"±"** —— 这是耦合电感的**最大难点**。正负号取决于两个线圈的**绕向**。

---

### 三、同名端——解决"±"问题的关键

#### 问题的由来

两个线圈的磁场有可能**相互加强**（磁通方向一致），也可能**相互削弱**（磁通方向相反）——取决于它们的**绕线方向**和**电流方向**。

#### 工程上的约定：用"同名端"标记

电路图上画两个线圈时，根本看不出绕线方向（就是两个曲线而已）。工程上在每个线圈的其中一端标一个"**·**"（黑点）：

```
     ·               ·
   ┌──[L₁]──┐     ┌──[L₂]──┐
   │  ∿∿∿   │     │  ∿∿∿   │
```

**同名端的物理意义**：

> **两个黑点端**：若两个电流都**从黑点端流入**，则它们产生的磁通**方向相同**（相互加强）。

#### 互感电压的正负号规则

**一句话口诀**：

> **"入同出同取正，入异出异取负"** —— 看两个电流相对于同名端的"进出"关系。

**更具体地**：对互感电压 $M \dfrac{di_2}{dt}$ 在 $u_1$ 方程中的正负号：

- 若 $i_2$ 从 L₂ 的同名端**流入**，并且 $u_1$ 的**"+"端在 L₁ 的同名端**：取 **+**
- 反之取 **−**

#### 举个实例

```
     ·  i₁→          ·  i₂→
   ┌──[L₁]──┐      ┌──[L₂]──┐
   │        │      │        │
   + u₁ -           + u₂ -
   
   黑点都在左上方，i₁、i₂都从黑点流入
   u₁、u₂的"+"端都在黑点端
```

此时：

$u_1 = L_1 \frac{di_1}{dt} + M \frac{di_2}{dt}$

$u_2 = L_2 \frac{di_2}{dt} + M \frac{di_1}{dt}$

（两个互感项都取 **+**，因为电流和黑点端的关系"入同出同"）

---

### 四、耦合系数 k

问题：互感 M 能不能无穷大？

**不能**。物理限制：**一个线圈的磁场，不可能全部都穿过另一个线圈**（总有漏磁）。

定义**耦合系数**：

$k = \frac{M}{\sqrt{L_1 L_2}}, \quad 0 \leq k \leq 1$

- $k = 0$：完全不耦合（两线圈离得远，磁场没交集）
- $k = 1$：**全耦合**（所有磁通都共享，没有漏磁），这是**理想变压器**的前提

所以 $M$ 有一个上界：$M \leq \sqrt{L_1 L_2}$

---

## 4.3 理想变压器

### 一、从耦合电感到理想变压器

变压器本质上就是一个**耦合电感**，只不过做到了**三个极端**：

1. **全耦合**：$k = 1$（没有漏磁）
2. **无损耗**：线圈电阻 = 0，铁芯无涡流损耗
3. **电感趋于无穷大**：$L_1, L_2, M \to \infty$，但匝数比 $N_1:N_2$ 保持有限

把这三个条件"极端化"，耦合电感的方程就退化成一个超级简洁的形式——**理想变压器**。

---

### 二、理想变压器的两个黄金方程

设一次侧（primary）匝数为 $N_1$，二次侧（secondary）匝数为 $N_2$，定义**匝数比** $n = N_1 / N_2$。

#### 电压变换

$\boxed{\frac{u_1}{u_2} = \frac{N_1}{N_2} = n}$

#### 电流变换

$\boxed{\frac{i_1}{i_2} = \frac{N_2}{N_1} = \frac{1}{n}}$

**注意方向约定**：标准电路图里 $i_1$ 从 1' 端流入一次侧，$i_2$ 从 2' 端流出二次侧（或反过来，取决于同名端位置）。这种约定下上面两个公式直接成立。

```
      i₁ →              ← i₂
   ┌──[ ]──┐         ┌──[ ]──┐
   │  ∿∿∿  │ ·       · │  ∿∿∿ │
   + u₁ -             + u₂ -
      N₁                N₂
```

---

### 三、理想变压器的三个关键性质

#### 性质 ①：输入功率 = 输出功率（无损耗）

$p_1 = u_1 i_1 = (n u_2)(i_2 / n) = u_2 i_2 = p_2$

**能量从一次侧瞬时、完整地传到二次侧**。这就是"理想"的真正含义。

#### 性质 ②：阻抗变换

这是理想变压器**最有用**的性质！

假设二次侧接一个负载 $Z_L$，则：$u_2 = Z_L i_2$

从一次侧看进去的等效阻抗：

$Z_{in} = \frac{u_1}{i_1} = \frac{n u_2}{i_2/n} = n^2 \cdot \frac{u_2}{i_2} = n^2 Z_L$

$\boxed{Z_{in} = n^2 Z_L}$

**一次侧看到的负载 = 实际负载 × 匝数比的平方**。

**工程应用**：这正是**阻抗匹配**的关键技术（第五章最大功率传输会再提）。

#### 性质 ③：注意它**不是**普通电感的极限

理想变压器**不储能**！它只是"瞬时"地把能量从一侧传到另一侧，不像电感那样储存磁场能。这是一个重要的概念区别。

---

### 四、举个简单例子

```
   ─[10V电源]──┤理想变压器 n=2:1├──[4Ω 负载]─
```

一次侧 10V，匝数比 2:1（即 n = 2）。

**二次侧电压**：$u_2 = u_1 / n = 10/2 = 5\,\text{V}$

**二次侧电流**：$i_2 = u_2 / 4 = 5/4 = 1.25\,\text{A}$

**一次侧电流**：$i_1 = i_2 / n = 1.25/2 = 0.625\,\text{A}$

**一次侧看到的等效阻抗**：$Z_{in} = n^2 \cdot 4 = 4 \cdot 4 = 16\,\Omega$

**验证**：一次侧 10V / 16Ω = 0.625 A ✓

**功率**：

- 一次侧：$10 \times 0.625 = 6.25\,\text{W}$
- 二次侧：$5 \times 1.25 = 6.25\,\text{W}$ ✓（相等，无损耗）

## 5.1 正弦量

### 一、为什么全世界的电都是正弦？

先聊个开放问题，你凭直觉回答：

**你家墙上的插座，电压是 220V（中国）或 110V（美国）。这个电压是恒定不变的吗？还是变化的？如果变化，怎么变？**

---

答案你可能知道：**是正弦变化的**，50Hz（中国）或 60Hz（美国）。

但为什么是正弦？不能是方波、三角波？原因很多（发电机原理、能量传输效率、变压器只对变化的量工作……），但最根本的一个数学原因是：

> **正弦是线性系统的"特征函数"**——正弦波经过任何线性电路（R、L、C 组合），出来还是**同频率**的正弦波，只是幅值和相位可能变了。

这是后面所有分析的基石。

---

### 二、正弦量的三要素

一个正弦电流（或电压）的完整数学形式：

$i(t) = I_m \cos(\omega t + \varphi)$

```
     i(t)
      ↑
   Im ┤    ╱╲              ╱╲
      │   ╱  ╲            ╱  ╲
      │  ╱    ╲          ╱    ╲
    0 ┼─────────────────────────→ t
      │        ╲        ╱
      │         ╲      ╱
  -Im ┤          ╲____╱
      │←  T  →│
```

**三个要素**：

|要素|符号|名称|单位|
|---|---|---|---|
|幅值|$I_m$|峰值/最大值|A|
|角频率|$\omega$|每秒转过的弧度|rad/s|
|初相|$\varphi$|t=0 时刻的相位|rad 或 °|

**三个要素之间的关系**：

$\omega = 2\pi f = \frac{2\pi}{T}$

- ff f：频率（每秒多少个周期），单位 Hz
- TT T：周期（一个周期的时长），单位 s

**例**：中国电网 f = 50 Hz，则 $\omega = 2\pi \times 50 = 314$ rad/s。你以后看到 314 这个数字别觉得眼熟是巧合——就是工频 50Hz 的角频率。

---

### 三、有效值（RMS）—— 一个重要概念

**思考题**：你家插座说"220V"，这个 220 是正弦的**峰值** $I_m$ 吗？

答案：**不是！** 220V 是**有效值**。

#### 有效值的定义（物理意义）

> **有效值 = 能在电阻上产生相同平均发热功率的直流电压/电流值。**

比如说，你家 220V 正弦电压给一个 100Ω 电阻供电，它平均发热功率是多少？

**用有效值算**：$P = U_{\text{有效}}^2 / R = 220^2 / 100 = 484\,\text{W}$

这就等于**用 220V 直流电压接同一个电阻的发热功率**。有效值的好处就是：**算平均功率时可以直接当直流用**。

#### 有效值与峰值的关系（对正弦）

经过数学推导（对 $\cos^2$ 在一个周期内求平均，取平方根）：

$U = \frac{U_m}{\sqrt{2}} \approx 0.707\,U_m$

所以 220V 的实际峰值是：$U_m = 220 \times \sqrt{2} \approx 311\,\text{V}$

**电气工程的默认约定**：说电压/电流时，**默认是有效值**。只有用“峰值”“幅值”这些词时才指 $U_m$。

---

### 四、相位差

两个**同频率**的正弦波之间可以比较"相位差"：

$u_1 = U_m \cos(\omega t + \varphi_1), \quad u_2 = U_m \cos(\omega t + \varphi_2)$

**相位差**：$\varphi = \varphi_1 - \varphi_2$

- $\varphi > 0$：$u_1$ **超前** $u_2$（$u_1$ 先达到峰值）
- $\varphi < 0$：$u_1$ **滞后** $u_2$
- $\varphi = 0$：**同相**
- $\varphi = \pm\pi$：**反相**

> ⚠️ **前提**：只有同频率的正弦波才能比相位差！不同频率比相位差是无意义的。

---

## 5.2 相量法基础 ⭐（本章核心中的核心）

现在我们要攻克"**复数表示波**"这个你的痛点。

**请你跟着我，我要慢慢把这件事讲清楚。** 这部分不着急，一步都不要跳。

---

### 一、先说说"为什么需要复数"—— 动机

假设现在有一个 RLC 串联电路，接到正弦电源 $u_s = U_m \cos \omega t$。用 KVL 列方程：

$u_s = Ri + L\frac{di}{dt} + \frac{1}{C}\int i \, dt$

这是一个**微分积分方程**。要求 $i(t)$，得解微分方程，很麻烦。

**有没有办法把这种方程变成普通的代数方程？**

有！这就是**相量法**的目的。它把**微积分方程**变成**代数方程**，代价是：**我们要用复数来表示正弦波**。

---

### 二、预备知识：复数的三种形式

快速复习一下（你说学过复数，应该不陌生）：

**代数形式**：$A = a + jb$（电路里用 $j$ 不用 $i$，因为 $i$ 被电流占了）

**几何形式**：$A$ 是复平面上的一个点（或向量）

```
        Im
         ↑
         │      • A = a + jb
       b ┤     ╱
         │    ╱
         │   ╱
         │  ╱
         │ ╱
         │╱
       0 ┼─────────→  Re
               a
```

**极坐标形式（模角式）**：$A = |A| \angle \theta$

- 模 $|A| = \sqrt{a^2 + b^2}$
- 辐角 $\theta = \arctan(b/a)$

**指数形式**（欧拉公式）：

$A = |A| e^{j\theta}$

#### 欧拉公式（⭐ 本章理论基石）

$\boxed{e^{j\theta} = \cos\theta + j\sin\theta}$

这个公式是**连接复数和三角函数的桥梁**。以后我们会反复用它。

---

### 三、把正弦波"升维"到复数（核心一跃）

现在最关键的一步来了。**请集中注意力**。

#### 第 1 步：观察一个奇妙的等式

考虑这个复指数：

$e^{j\omega t}$

由欧拉公式：$e^{j\omega t} = \cos(\omega t) + j\sin(\omega t)$

取它的**实部**：

$\operatorname{Re}\left[e^{j\omega t}\right] = \cos(\omega t)$

**发现了吗**：一个正弦波 $\cos(\omega t)$，可以写成旋转复指数 $e^{j\omega t}$ 的**实部**！

#### 第 2 步：一般的正弦波

对 $i(t) = I_m \cos(\omega t + \varphi)$：

$i(t) = I_m \operatorname{Re}\left[e^{j(\omega t + \varphi)}\right] = \operatorname{Re}\left[I_m e^{j\varphi} \cdot e^{j\omega t}\right]$

中间的 $I_m e^{j\varphi}$ 是一个**与时间无关**的常数（复数），只含幅值信息 $I_m$ 和相位信息 $\varphi$。

$e^{j\omega t}$ 这一部分包含“时间”和“角频率”，但在同频率电路里它对每个量都一样。

#### 第 3 步：相量的定义

既然在同频率电路里，$e^{j\omega t}$ 对所有量都一样，我们就**把它“省略”**，只保留含信息的那个复数常数，这就叫**相量**（phasor）：

$\boxed{\dot{I} = I_m e^{j\varphi} = I_m \angle \varphi}$

> **相量**是一个**复数**，含有正弦波的幅值和初相信息。频率 $\omega$ 不在相量里（因为同电路里都一样）。

顶上的小点（或箭头）表示"这是相量，不是普通复数"。

#### 第 4 步：有效值相量

工程上常用**有效值相量**（把幅值换成有效值）：

$\dot{I} = I \angle \varphi = \frac{I_m}{\sqrt{2}} \angle \varphi$

本课程之后**默认用有效值相量**，除非特别说明。

---

### 四、相量的物理图像：旋转的向量

想象复平面上有一个向量，长度 $I_m$，初始角度 $\varphi$。让它以角速度 $\omega$ **逆时针旋转**：

```
   时刻 t=0                 时刻 t>0（转了 ωt）
   
      Im                        Im
       ↑                          ↑
       │ • I_m∠φ                  │    •←旋转向量
       │╱                         │   ╱
       │                          │  ╱
     0 ┼─→  Re                  0 ┼─────→ Re
                                  │
                                  │ 投影到实轴就是 i(t)
                                  ↓
                                cos(ωt+φ)
```

**关键图像**：

- 旋转向量的**实部投影** = 时域正弦波
- 向量的**长度** = 正弦波的幅值
- 向量**当前的辐角** = 当前的相位（$\omega t + \varphi$）

**相量就是"抓拍" t=0 时刻的那个向量**。因为所有量都以同样的 $\omega$ 旋转，它们的**相对关系**不变——抓拍一瞬间就够了。

---

### 五、相量法为什么能"化微积分为代数"？

这是相量法的威力所在。我现在演示给你看。

#### 关键技巧：微分 → 乘 jω

假设 $i(t) = I_m \cos(\omega t + \varphi)$，其相量为 $\dot{I} = I_m \angle \varphi$。

求导：

$\frac{di}{dt} = -\omega I_m \sin(\omega t + \varphi) = \omega I_m \cos(\omega t + \varphi + 90^\circ)$

这个导数对应的相量是：$\omega I_m \angle(\varphi + 90^\circ)$

而 $\omega I_m \angle(\varphi + 90^\circ) = I_m \angle\varphi \cdot \omega \angle 90^\circ = \dot{I} \cdot j\omega$

（因为 $\angle 90^\circ = \cos 90^\circ + j\sin 90^\circ = j$）

**结论**：

$\boxed{\frac{di}{dt} \quad \longleftrightarrow \quad j\omega \dot{I}}$

**时域求导 = 相量域乘 jω**！

#### 类似地：积分 → 除以 jω

$\boxed{\int i \, dt \quad \longleftrightarrow \quad \frac{\dot{I}}{j\omega}}$

#### 这意味着什么？

回到最开始的 RLC 串联微积分方程：

$u_s = Ri + L\frac{di}{dt} + \frac{1}{C}\int i \, dt$

**相量化**（每一项替换）：

$\dot{U}_s = R\dot{I} + j\omega L \dot{I} + \frac{1}{j\omega C}\dot{I}$

提取 $\dot{I}$：

$\dot{U}_s = \left[R + j\omega L + \frac{1}{j\omega C}\right]\dot{I}$

**这是一个代数方程！** 没有求导、没有积分。方括号里那一坨复数叫"阻抗"（下一节详细讲）。

---

### 六、消化一下——你需要记住的核心

现在停下来，把这几件事"钉"在脑子里：

|时域|对应|相量域|
|---|---|---|
|$i(t) = I_m\cos(\omega t + \varphi)$||$\dot{I} = I\angle\varphi$（有效值）|
|$\dfrac{di}{dt}$||$j\omega \dot{I}$|
|$\int i \, dt$||$\dfrac{\dot{I}}{j\omega}$|
|微分方程||代数方程|

**整个相量法的"交易"**：

> 我用**复数**这个更复杂的数，换来了**代数方程**这个更简单的方程。

对于正弦稳态分析，这笔交易**非常划算**。

## 5.3 电路元件的相量形式

前面我们把"正弦波"翻译成了"相量"，把"微积分"翻译成了"乘除 jω"。现在要做一件事：把电阻、电感、电容这三个基本元件的 VCR，也**全部翻译成相量形式**。

翻译完之后你会看到一个奇妙的结果：**三种完全不同的元件，将拥有同一种数学形式**。这就是相量法的真正威力。

---

### 一、电阻 R 的相量形式

时域 VCR：$u = Ri$

假设 $i(t)=I_m \cos(\omega t + \varphi)$，则 $u(t)=RI_m \cos(\omega t + \varphi)$。

**观察**：电压和电流**同频率、同相位**，幅值差 R 倍。

转成相量：

$\boxed{\dot{U} = R\dot{I}}$

**这就是相量形式的"欧姆定律"**，和时域形式一模一样——电阻是最朴素的，不改变相位。

**相量图**（复平面上表示）：

```
         Im
          ↑
          │     
          │        → U = RI  (同方向，U 更长)
          │     
          │      → I
          │     
        0 ┼────────────→  Re
```

$\dot{U}$ 和 $\dot{I}$ 是**同向**的两个复向量。

---

### 二、电感 L 的相量形式 ⭐

时域 VCR：$u = L \dfrac{di}{dt}$

用我们刚学的"微分 → 乘 jω"规则：

$\boxed{\dot{U} = j\omega L \cdot \dot{I}}$

**这里面藏着两个重要信息**，请你仔细品：

---

**请你先思考**：$j\omega L$ 是一个复数，它的**模**和**辐角**分别是什么？它对 $\dot{I}$ 做了什么“操作”？

---

答案：

- **模**：$|j\omega L| = \omega L$
- **辐角**：$\angle(j\omega L) = 90^\circ$（因为 $j = 1\angle 90^\circ$）

所以 $\dot{U} = (\omega L)\angle 90^\circ \cdot \dot{I}$，意思是：

> **对电感：电压的幅值 = 电流幅值 × $\omega L$；电压的相位比电流超前 $90^\circ$**

**这正是电感的物理本质**！电感"反抗电流变化"的特性，在相量图上就是"电压领先电流 90°"。

**相量图**：

```
         Im
          ↑
          │        → U = jωL·I  (垂直方向，超前 90°)
          │        │
          │        │
          │        │
          │     → I
        0 ┼────────────→  Re
```

**记忆口诀**：电感"**电压超前**"。

---

### 三、电容 C 的相量形式 ⭐

时域 VCR：$i = C \dfrac{du}{dt}$

相量化：$\dot{I} = j\omega C \cdot \dot{U}$

整理成 "电压 = ? × 电流" 的形式：

$\boxed{\dot{U} = \frac{1}{j\omega C} \dot{I} = \frac{-j}{\omega C}\dot{I}}$

（用到 $1/j = -j$，因为 $j \cdot (-j) = -j^2 = 1$）

**分析**：

- **模**：$\left|\dfrac{1}{j\omega C}\right| = \dfrac{1}{\omega C}$
- **辐角**：$\angle\left(\dfrac{1}{j\omega C}\right) = -90^\circ$

所以：

> **对电容：电压幅值 = 电流幅值 × $\dfrac{1}{\omega C}$；电压相位比电流滞后 $90^\circ$**

**相量图**：

```
         Im
          ↑
          │     → I
          │     
        0 ┼────────────→  Re
          │     │
          │     │
          │     │
          │     ↓ U  (超后 90°)
```

**记忆口诀**：电容"**电流超前**"。

---

### 四、三个元件对比总结

请务必把下面这张表**刻在脑子里**：

|元件|时域 VCR|相量 VCR|$\dot{U}$ 与 $\dot{I}$ 的相位关系|
|---|---|---|---|
|电阻 R|$u = Ri$|$\dot{U} = R\dot{I}$|同相|
|电感 L|$u = L\,di/dt$|$\dot{U} = j\omega L\,\dot{I}$|$\dot{U}$ 超前 $\dot{I}$ $90^\circ$|
|电容 C|$i = C\,du/dt$|$\dot{U} = \dfrac{1}{j\omega C}\dot{I}$|$\dot{U}$ 滞后 $\dot{I}$ $90^\circ$|

---

**一个非常关键的观察**：三个元件的相量 VCR，**都长得像欧姆定律**！

$\dot{U} = Z \cdot \dot{I}$

只是那个"Z"对不同元件不同：

- 电阻：$Z = R$
- 电感：$Z = j\omega L$
- 电容：$Z = \dfrac{1}{j\omega C}$

这个 Z 就叫**阻抗**——下一节的主角。

---

## 5.4 阻抗与导纳

### 一、阻抗的定义

> **阻抗**（Impedance）$Z$：在相量域，一个二端元件（或网络）的电压相量与电流相量之比。
> 
> $Z = \dfrac{\dot{U}}{\dot{I}}$

单位：**欧姆 Ω**（和电阻相同）。

**注意**：阻抗是一个**复数**，不是时域的物理量（不要试图写 Z(t)Z(t) Z(t)）。

---

### 二、阻抗的几种形式

**代数形式**：

$Z = R + jX$

其中：

- $R$：**电阻**分量（实部）
- $X$：**电抗**分量（虚部），单位也是欧姆
- $X>0$：**感性**（电感为主）
- $X<0$：**容性**（电容为主）

**极坐标形式**：

$Z = |Z| \angle \varphi_Z$

- $|Z| = \sqrt{R^2 + X^2}$：**阻抗模**
- $\varphi_Z = \arctan(X/R)$：**阻抗角**

**阻抗角的物理意义**：电压相位超前电流相位的角度。

$\varphi_Z = \varphi_u - \varphi_i$

---

### 三、三个基本元件的阻抗

**电阻**：$Z = R$（纯实数）

**电感**：$Z = j\omega L$（纯虚数，正虚部）

- 电感的电抗 $X_L = \omega L$，称为**感抗**
- **频率越高，感抗越大**（电感对高频的阻碍越强）

**电容**：$Z = \dfrac{1}{j\omega C} = -\dfrac{j}{\omega C}$（纯虚数，负虚部）

- 电容的电抗 $X_C = -\dfrac{1}{\omega C}$，称为**容抗**
- **频率越高，容抗越小**（电容对高频的阻碍越弱）

---

### 四、阻抗的直观理解

**感抗 $X_L = \omega L$ 随频率变化**：

```
  X_L
   ↑
   │      ╱
   │    ╱
   │  ╱
   │╱
 0 ┼────────→ ω
```

**容抗 $|X_C| = 1/(\omega C)$ 随频率变化**：

```
  |X_C|
   ↑
   │╲
   │ ╲
   │  ╲____
   │       ─────
 0 ┼────────→ ω
```

**两个重要极限**：

- $\omega \to 0$（直流）：$Z_L \to 0$（短路），$Z_C \to \infty$（开路），这和第四章结论一致。
- $\omega \to \infty$（高频）：$Z_L \to \infty$（开路），$Z_C \to 0$（短路）。

**工程解读**：

- 电感"喜欢直流，讨厌高频"——常用来做**扼流圈**（阻止高频干扰）
- 电容"讨厌直流，喜欢高频"——常用来做**耦合电容**（隔直流、通交流）

---

### 五、阻抗的串并联（和电阻完全一样！）

这是相量法最妙的地方——**阻抗的串并联公式，和电阻的串并联公式形式完全相同**，只不过把实数换成复数：

**串联**：$Z_{\text{总}} = Z_1 + Z_2 + \cdots$

**并联**：$\dfrac{1}{Z_{\text{总}}} = \dfrac{1}{Z_1} + \dfrac{1}{Z_2} + \cdots$

（两个阻抗并联：$Z_{\text{总}} = \dfrac{Z_1 Z_2}{Z_1 + Z_2}$）

**分压、分流公式也完全一样**，只是用复数运算。

> 💡 这意味着：第二章学的所有分析方法（节点法、回路法）、第三章学的所有定理（戴维南、叠加……），**在正弦稳态下全部适用**，只要把"电阻"换成"阻抗"，把"电压电流"换成"相量"即可。

---

### 六、导纳 Y

**导纳**（Admittance）是阻抗的倒数：

$Y = \dfrac{1}{Z} = \dfrac{\dot{I}}{\dot{U}}$

单位：**西门子 S**

**代数形式**：

$Y = G + jB$

- $G$：**电导**（Conductance）
- $B$：**电纳**（Susceptance）

**重要陷阱**⚠️：$G \neq 1/R$，$B \neq 1/X$！正确的转换是：

$Y = \frac{1}{Z} = \frac{1}{R+jX} = \frac{R-jX}{R^2+X^2}$

所以 $G = \dfrac{R}{R^2+X^2}$，$B = \dfrac{-X}{R^2+X^2}$

**什么时候用导纳？**

- 分析**并联**电路时，导纳相加更方便（因为 $Y_{\text{总}} = Y_1 + Y_2 + \cdots$）
- 用**节点法**分析正弦电路时，导纳是主角

### 🎯 一道综合例题

来一道题检验你对相量、阻抗的掌握程度。**这是本章最重要的一类题**——把前面所有概念串起来。

---

**题目**：

一个 RLC 串联电路：$R = 10\,\Omega$，$L = 0.1\,\text{H}$，$C = 100\,\mu\text{F}$，接到正弦电压源 $u_s(t) = 100\sqrt{2}\cos(100t)\,\text{V}$ 上。

**求**：

1. 电源相量 $\dot{U}_s$
2. 电路总阻抗 $Z$
3. 电路电流相量 $\dot{I}$
4. 电流的时域表达式 $i(t)$

---

**提示步骤**：

1. **写电源相量**：幅值 $100\sqrt{2}$ 对应有效值 100，初相为 $0^\circ$。所以 $\dot{U}_s = 100 \angle 0^\circ\,\text{V}$。
2. **算各元件阻抗**（$\omega = 100\,\text{rad/s}$）：
    - $Z_R = 10\,\Omega$
    - $Z_L = j\omega L = j \cdot 100 \cdot 0.1 = j10\,\Omega$
    - $Z_C = \dfrac{1}{j\omega C} = \dfrac{1}{j \cdot 100 \cdot 100\times 10^{-6}} = \dfrac{1}{j \cdot 0.01} = \dfrac{-j}{0.01} = -j100\,\Omega$
3. **求总阻抗**（串联相加）：
    - $Z = Z_R + Z_L + Z_C = 10 + j10 - j100 = 10 - j90\,\Omega$
    - 化成极坐标：$|Z| = \sqrt{10^2 + 90^2} = \sqrt{8200} \approx 90.55\,\Omega$
    - $\varphi_Z = \arctan(-90/10) = \arctan(-9) \approx -83.66^\circ$
    - 所以 $Z \approx 90.55 \angle -83.66^\circ\,\Omega$（**容性**，因为虚部为负）
4. **求电流相量**（欧姆定律）：
    - $\dot{I} = \dot{U}_s / Z = \dfrac{100\angle 0^\circ}{90.55\angle -83.66^\circ} = \dfrac{100}{90.55}\angle(0^\circ - (-83.66^\circ)) \approx 1.10 \angle 83.66^\circ\,\text{A}$
5. **转回时域**：
    - 电流有效值 1.10 A，所以幅值 $I_m = 1.10\sqrt{2} \approx 1.56\,\text{A}$
    - 初相 $83.66^\circ$
    - $i(t) \approx 1.56 \cos(100t + 83.66^\circ)\,\text{A}$

---

**物理解读**：

- 电路是**容性的**（$X_C > X_L$），所以**电流超前电压** $83.66^\circ$（这和纯电容“电流超前 $90^\circ$”接近，因为电容的作用远大于电感）
- 阻抗模 90.55 Ω 中，电阻只占 10 Ω，所以大部分是电抗（无功储能），这也是电流超前角接近 90° 的原因
