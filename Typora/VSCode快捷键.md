1. **光标与选择**

- **`Ctrl + L`**：选中当前整行；在终端中可用于清屏。
- **`Ctrl + D`**：选中当前光标下的单词，再次按下可继续选中下一个相同单词。
- **`Ctrl + Shift + L`**：一次性选中所有与当前选中内容相同的词。
- **`Ctrl + Alt + ↑ / ↓`**：在当前光标上方或下方添加光标，进行**多光标编辑**。

2. **行操作与基础编辑**

- **`Alt + ↑ / ↓`**：将当前行或选中的代码块向上或向下移动。     
- **`Alt + Shift + ↑ / ↓`**：向上或向下复制当前行或选中的代码块。
- **`Ctrl + Enter`**：在下方新起一行插入。
- **`Ctrl + Shift + Enter`**：在上方新起一行插入。
- **`Ctrl + Shift + K`**：删除当前整行。
- **`Ctrl + /`**：对当前行或选中内容进行**注释/取消注释**。
- **`Alt + Shift + A`**：添加或取消**块注释**（如 `/* ... */`）。
- **`Ctrl + K, Ctrl + X`**：清除行尾空格。
- **`Shift + Alt + F`**：格式化当前文件。

3. **导航、搜索与代码查看**

- **`Ctrl + G`**：跳转到指定行号。
- **`Ctrl + P`**：快速搜索并跳转到文件。
- **`Ctrl + F`**：在当前文件中查找。
- **`Ctrl + H`**：在当前文件中替换。
- **`F12`**：跳转到定义。
- **`Alt + F12`**：预览定义，不离开当前位置。
- **`Ctrl + K, Ctrl + I`**：显示悬浮提示。
- **`Ctrl + Space`**：强制触发代码补全提示。
- **`Ctrl + .`**：打开**快速修复**建议。
- **`Ctrl + K, V`**：在侧边栏预览 Markdown。
- **`Ctrl + K, D`**：在资源管理器中定位当前文件。

4. **重构与代码折叠**

- **`F2`**：重命名变量、函数等符号，并同步更新引用。
- **`Ctrl + K, Ctrl + 0`**：**折叠全部**代码块。
- **`Ctrl + K, Ctrl + J`**：**展开全部**代码块。
- **`Ctrl + K, Ctrl + [`**：递归折叠当前区域。
- **`Ctrl + K, Ctrl + ]`**：递归展开当前区域。
- **`Ctrl + K, Ctrl + 2`**：折叠所有二级区块；同理可使用 **`Ctrl + K, Ctrl + 1`** 到 **`Ctrl + K, Ctrl + 7`** 折叠对应级别。

5. **编辑器与窗口管理**

- **`Ctrl + K, Enter`**：固定当前编辑器标签页。
- **`Ctrl + K, Ctrl + ← / →`**：将当前编辑器移动到左侧或右侧分屏。
- **`Ctrl + \`**：分屏。
- **`Alt + 数字键`**：在不同的编辑器标签页之间快速切换。
- **`Ctrl + Alt + ← / →`**：在不同的编辑器分组之间快速切换。
- **`Ctrl + W`**：关闭当前编辑器标签页或窗口。
- **`Ctrl + K, W`**：关闭所有已保存的文件。
- **`Ctrl + J`**：显示或隐藏底部面板。
- **`Ctrl + K, Z`**：进入**禅模式**。

6. **工作区与侧边栏面板**

- **`Ctrl + Shift + E`**：聚焦到**资源管理器**。
- **`Ctrl + B`**：显示或隐藏**侧边栏**。
- **`Ctrl + Shift + F`**：打开**全局搜索**。
- **`Ctrl + Shift + G`**：打开**源代码管理 / Git** 面板。
- **`Ctrl + Shift + D`**：打开**调试**面板。
- **`Ctrl + Shift + X`**：打开**扩展**面板。

7. **设置、语言与文件操作**

- **`Ctrl + K, Ctrl + S`**：打开快捷键设置。
- **`Ctrl + K, S`**：**全部保存**。
- **`Ctrl + K, M`**：更改当前文件的语言模式。
